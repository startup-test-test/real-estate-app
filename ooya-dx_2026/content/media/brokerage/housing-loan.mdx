## 住宅ローンに仲介手数料を含められる？結論から解説

不動産を購入するとき、仲介手数料は数十万円〜100万円以上にもなります。「この金額を住宅ローンに含められないか？」と考える方は多いでしょう。

### 結論：原則NG、ただし方法はある

住宅ローンは**「物件の購入費用」**に充てるためのものです。仲介手数料はあくまで不動産会社へのサービス報酬のため、**原則として住宅ローンの融資対象に含まれません**。

しかし、**諸費用ローン**を利用すれば仲介手数料を含めた借り入れが可能です。

| 方法 | 仲介手数料の扱い |
|------|----------------|
| 住宅ローンのみ | **対象外**（組み込めない） |
| 住宅ローン＋諸費用ローン | **対象**（組み込める） |
| オーバーローン（一部銀行） | **対象**（条件あり） |

## 【体験談】実際にローンに組み込んだ流れ

筆者自身、住宅購入時に仲介手数料を諸費用ローンに組み込んだ経験があります。その実体験をお伝えします。

### きっかけ：手元資金を残したかった

住宅を購入する際、物件の頭金と引越し費用で手元資金がかなり減る見込みでした。仲介手数料（約100万円）まで現金で支払うと、**生活防衛資金が不足する**ことが分かり、ローンへの組み込みを検討しました。

### 銀行とのやり取り

1. **住宅ローンの仮審査時**に、諸費用ローンの利用を相談
2. 銀行から「諸費用ローンは住宅ローンとセットで審査可能」との回答
3. **必要書類**：仲介手数料の見積書（不動産会社から取得）
4. 諸費用ローンの金利は住宅ローンより**約2%高い**と説明を受ける
5. 返済期間は住宅ローンと同じ35年で設定可能だが、**10年で組んだ**

### 実際にかかった手間

正直に言うと、**手続き自体は簡単**でした。住宅ローンの申込書類に加え、諸費用の明細書を提出するだけ。審査も住宅ローンと同時に行われたため、追加の手間はほとんどありませんでした。

### 振り返って思うこと

- 手元資金を残せたのは**精神的な安心感**につながった
- ただし金利差を考えると、**余裕があれば現金払いが有利**だった
- 10年で完済したが、35年にしていたら利息がかなり膨らんでいた

## 諸費用ローンとは？金利・審査の違い

### 諸費用ローンの概要

諸費用ローンは、住宅購入に伴う以下の費用を借り入れるためのローンです。

| 対象となる諸費用 |
|----------------|
| **仲介手数料** |
| 登記費用（登録免許税＋司法書士報酬） |
| 火災保険料 |
| 印紙税 |
| 住宅ローン事務手数料 |
| 引越し費用（一部銀行） |

### 住宅ローンとの金利差

| ローン種別 | 金利の目安（変動金利） |
|-----------|---------------------|
| 住宅ローン | 年0.3〜0.7%程度 |
| 諸費用ローン | 年2〜4%程度 |

※金利は金融機関・時期・審査結果により異なります。最新の条件は各金融機関にご確認ください。

金利差は**約1.5〜3.5%**。仲介手数料100万円を諸費用ローンで35年返済すると、利息だけで**数十万円**上乗せされます。

### 審査の違い

| 項目 | 住宅ローン | 諸費用ローン |
|------|-----------|-------------|
| 審査基準 | 物件価値＋返済能力 | 返済能力重視 |
| 担保 | 物件に抵当権 | 無担保 or 住宅ローンに付随 |
| 借入上限 | 物件価格の100%程度 | 諸費用の実費分 |
| 審査の厳しさ | 標準 | やや厳しい場合あり |

諸費用ローンを住宅ローンとセットで申し込む場合、**追加の審査なし**で同時に通ることが多いです。ただし、借入総額が増えるため、**返済負担率**の面で不利になる可能性があります。

## 組み込む場合のメリット・デメリット

### メリット

| メリット | 詳細 |
|---------|------|
| **手元資金を残せる** | 生活防衛資金や引越し費用に充てられる |
| **初期費用を抑えられる** | まとまった現金がなくても購入可能 |
| **資金計画が立てやすい** | 月々の返済に組み込むことで管理しやすい |

### デメリット

| デメリット | 詳細 |
|-----------|------|
| **金利が高い** | 住宅ローンの数倍の金利がかかる |
| **総支払額が増える** | 利息分だけ余計に支払うことになる |
| **審査が厳しくなる可能性** | 借入総額が増えるため |
| **返済期間が長くなる** | 35年で組むと利息が膨らむ |

### 具体的な比較

仲介手数料100万円の場合：

| 支払い方法 | 支払い総額 |
|-----------|-----------|
| 現金一括 | **100万円** |
| 諸費用ローン（金利3%・10年） | 約116万円 |
| 諸費用ローン（金利3%・35年） | 約157万円 |

**手元資金に余裕があるなら、現金払いが最も有利**です。

最もシンプルな方法です。仲介手数料は一般的に**契約時に50%、決済時に50%**を支払います。

## よくある質問（FAQ）

### 仲介手数料は住宅ローンに組み込める？

原則対象外ですが、諸費用ローンで組み込めます。筆者の実体験は上記の体験談セクションで詳しく解説しています。

### 諸費用ローンとは？

住宅購入の諸費用（仲介手数料、登記費用など）を借り入れるローンです。住宅ローンとセットで申し込めます。

### 諸費用ローンの金利はどれくらい？

年2〜4%程度。住宅ローン（0.3〜0.7%程度）よりかなり高いです。

### 仲介手数料をローンに組み込むデメリットは？

金利が高い・総支払額増加・審査が厳しくなる可能性。手元資金に余裕があれば現金払いが有利です。

### ローンに組み込めない場合はどうする？

現金一括、仲介手数料の **[仲介手数料の交渉・値引き方法｜言い方・タイミング・成功する例文を解説【2026年版】](/media/brokerage/negotiation/)**、売主直売物件を探す、などの方法があります。
