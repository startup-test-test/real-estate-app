## 仲介手数料を住宅ローンに含めて審査までの流れ

著者自身、住宅ローンの融資審査ですが、金融機関は1社だけではなく、**2社の事前審査**を行っております。

物件価格と諸費用込みでの住宅ローンの審査結果として、

### 1. 大手メガバンク → 諸費用はNG。物件価格のみ。

理由として、**物件評価額までしか融資は出せません**との事でしたので、諸費用はNGでした。

### 2. 地元の信用金庫 → OK

諸費用込みで、仲介手数料や、火災保険料、司法書士手数料なども、OKとの事でしたので、**地元の信用金庫にて、住宅ローンを組むことを決意**致しました。

### 実際の流れ

1. **住宅ローンの仮審査時**に、諸費用込みのオーバーローンを相談
2. 対応可能な金融機関を確認（すべての銀行が対応しているわけではない）。著者の実体験としては、**地元の信用金庫などが積極的**なイメージです。
3. **必要書類**：仲介手数料の見積書・司法書士の見積書・引越し見積書など
4. 住宅ローンと同じ金利で、諸費用もまとめて借り入れ
5. 審査は通常の住宅ローンと同時に実施

### 振り返って思うこと

- 住宅ローンと同じ低金利で借りられたのが**最大のメリット**
- 手元資金を残せたのは**今後の安心感**につながった
- ただし、オーバーローンに対応する金融機関は限られるので、**事前に複数行に相談するのがポイント**

## 住宅ローンに諸費用も組み込むメリット・デメリット

### メリット

| メリット | 詳細 |
|---------|------|
| **手元資金を残せる** | 生活防衛資金や引越し費用に充てられる |
| **初期費用を抑えられる** | まとまった現金がなくても購入可能 |
| **資金計画が立てやすい** | 月々の返済に組み込むことで管理しやすい |

### デメリット

| デメリット | 詳細 |
|-----------|------|
| **金利が高い** | 住宅ローンの数倍の金利がかかる |
| **総支払額が増える** | 利息分だけ余計に支払うことになる |
| **審査が厳しくなる可能性** | 借入総額が増えるため |
| **返済期間が長くなる** | 35年で組むと利息が膨らむ |

### 具体的な比較

仲介手数料100万円の場合：

| 支払い方法 | 支払い総額 |
|-----------|-----------|
| 現金一括 | **100万円** |
| 諸費用ローン（金利3%・10年） | 約116万円 |
| 諸費用ローン（金利3%・35年） | 約157万円 |

**手元資金に余裕があるなら、現金払いが最も有利**です。

## 諸費用ローンという選択肢もあった

筆者はオーバーローンを選びましたが、もう一つの方法として「諸費用ローン」があります。筆者も諸費用ローンの存在は知っていましたが、**金利が住宅ローンより高い**ため、オーバーローンを選択しました。

### 諸費用ローンの概要

諸費用ローンは、住宅購入に伴う以下の費用を借り入れるためのローンです。

| 対象となる諸費用 |
|----------------|
| **仲介手数料** |
| 登記費用（登録免許税＋司法書士報酬） |
| 火災保険料 |
| 印紙税 |
| 住宅ローン事務手数料 |
| 引越し費用（一部銀行） |

### 住宅ローンとの金利差

| ローン種別 | 金利の目安（変動金利） |
|-----------|---------------------|
| 住宅ローン | 年0.3〜0.7%程度 |
| 諸費用ローン | 年2〜4%程度 |

※金利は金融機関・時期・審査結果により異なります。最新の条件は各金融機関にご確認ください。

金利差は**約1.5〜3.5%**。仲介手数料100万円を諸費用ローンで35年返済すると、利息だけで**数十万円**上乗せされます。

### 審査の違い

| 項目 | 住宅ローン | 諸費用ローン |
|------|-----------|-------------|
| 審査基準 | 物件価値＋返済能力 | 返済能力重視 |
| 担保 | 物件に抵当権 | 無担保 or 住宅ローンに付随 |
| 借入上限 | 物件価格の100%程度 | 諸費用の実費分 |
| 審査の厳しさ | 標準 | やや厳しい場合あり |

諸費用ローンを住宅ローンとセットで申し込む場合、**追加の審査なし**で同時に通ることが多いです。ただし、借入総額が増えるため、**返済負担率**の面で不利になる可能性があります。

## よくある質問（FAQ）

### 仲介手数料は住宅ローンに組み込める？

原則対象外ですが、諸費用ローンで組み込めます。筆者の実体験は上記の体験談セクションで詳しく解説しています。

### 諸費用ローンとは？

住宅購入の諸費用（仲介手数料、登記費用など）を借り入れるローンです。住宅ローンとセットで申し込めます。

### 諸費用ローンの金利はどれくらい？

年2〜4%程度。住宅ローン（0.3〜0.7%程度）よりかなり高いです。

### 仲介手数料をローンに組み込むデメリットは？

金利が高い・総支払額増加・審査が厳しくなる可能性。手元資金に余裕があれば現金払いが有利です。

### ローンに組み込めない場合はどうする？

現金一括、仲介手数料の **[仲介手数料の交渉・値引き方法｜言い方・タイミング・成功する例文を解説【2026年版】](/media/brokerage/negotiation/)**、売主直売物件を探す、などの方法があります。

## まとめ

仲介手数料の住宅ローンへの組み込みは、知っておくべき選択肢の一つです。

- **原則**：住宅ローンの対象外
- **方法①オーバーローン**：住宅ローンに諸費用を上乗せ。金利は住宅ローンと同じ低金利
- **方法②諸費用ローン**：別枠で借り入れ。金利は住宅ローンより2〜3%高い
- **メリット**：手元資金を残せる
- **注意点**：オーバーローン対応の金融機関は限られるため、事前確認が必要
- **体験談**：筆者はオーバーローンで仲介手数料・司法書士費用・引越し代をすべて組み込んで購入
