## NOI（営業純収益）とは

**NOI（Net Operating Income：営業純収益）**は、不動産から得られる収入から、運営にかかる経費を差し引いた純粋な営業利益のことです。

**借入の返済や税金を含まない**点が特徴で、物件そのものの収益力を評価できます。

---

## NOIの計算式

```
NOI = 総収入 − 運営経費
```

### 総収入に含まれるもの
- 家賃収入
- 共益費・管理費収入
- 駐車場収入
- その他収入（自動販売機など）

### 運営経費に含まれるもの
- 管理費・PM費
- 修繕費
- 水道光熱費（共用部）
- 固定資産税・都市計画税
- 保険料
- 空室損失

### 運営経費に含まれないもの
- ローン返済（元金・利息）
- 減価償却費
- 所得税・法人税

## NOIの計算例

| 項目 | 金額 |
|------|------|
| 年間家賃収入 | 600万円 |
| 共益費収入 | 60万円 |
| 空室損失（5%） | ▲33万円 |
| **総収入** | **627万円** |
| 管理費（5%） | ▲30万円 |
| 修繕費 | ▲30万円 |
| 固定資産税 | ▲50万円 |
| 保険料 | ▲10万円 |
| **運営経費** | **▲120万円** |
| **NOI** | **507万円** |

## NOIとNCF（ネットキャッシュフロー）の違い

| 指標 | 計算 | 用途 |
|------|------|------|
| NOI | 収入 − 運営経費 | 物件の収益力評価 |
| NCF | NOI − 資本的支出 | より保守的な評価 |

NCFは大規模修繕などの**資本的支出（CapEx）**を考慮した指標です。

---

## NOIの活用シーン

### 1. キャップレートの計算

```
キャップレート = NOI ÷ 物件価格 × 100（%）
```

NOIを使って物件の**還元利回り**を計算できます。

### 2. 物件価格の算出（収益還元法）

```
物件価格 = NOI ÷ キャップレート
```

NOIとエリアのキャップレートから、**適正価格**を逆算できます。

### 3. 物件比較

借入条件に左右されないため、**異なる物件の収益力を公平に比較**できます。

---

## NOIの目安（NOI利回り）

| NOI利回り | 評価 |
|----------|------|
| 3%未満 | 低い（都心一等地など） |
| 3〜5% | 標準的（都市部） |
| 5〜7% | やや高い（地方都市） |
| 7%以上 | 高い（築古・地方） |
