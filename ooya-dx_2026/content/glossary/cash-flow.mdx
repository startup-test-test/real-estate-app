## キャッシュフロー（CF）とは

**キャッシュフロー（CF：Cash Flow・現金収支）**は、実際に手元に入ってくるお金と出ていくお金の流れを表します。

簡単に言うと、**「実際に手元にいくら残るか」**を表す指標です。

---

## なぜキャッシュフローが重要なのか

| 観点 | 説明 |
|------|------|
| 実際の手残り | 帳簿上の利益ではなく実際の現金 |
| 資金繰り | ローン返済や経費支払いの原資 |
| 経営の安定性 | プラスを維持できるかが重要 |

利回りが高くても、キャッシュフローがマイナスになると資金繰りが厳しくなります。

---

## キャッシュフローの種類

賃貸経営では、以下の3段階でキャッシュフローを把握します：

| 種類 | 計算内容 | 意味 |
|------|---------|------|
| NOI | 賃料 - 経費 | 営業純収益 |
| 税引前CF | NOI - ローン返済 | 税金を払う前の手残り |
| 税引後CF | 税引前CF - 税金 | 最終的な手残り |

---

## キャッシュフローの計算式

### NOI（営業純収益）
```
NOI = 年間賃料収入 - 空室損失 - 運営経費
```

### 税引前キャッシュフロー
```
税引前CF = NOI - 年間ローン返済額（ADS）
```

### 税引後キャッシュフロー
```
税引後CF = 税引前CF - 所得税・住民税
```

---

## キャッシュフローの計算例

| 項目 | 金額 |
|------|------|
| 年間賃料収入 | 360万円 |
| 空室損失（5%） | 18万円 |
| 運営経費 | 72万円 |
| 年間ローン返済 | 180万円 |

```
NOI = 360万円 - 18万円 - 72万円 = 270万円
税引前CF = 270万円 - 180万円 = 90万円（月7.5万円）
```

---

## キャッシュフローに影響する要因

| 要因 | 影響 |
|------|------|
| 空室率 | 高いほどCFが減少 |
| 賃料水準 | 下落するとCFが減少 |
| 金利 | 上昇するとローン返済が増加 |
| 経費 | 修繕費等の増加でCFが減少 |
| 借入期間 | 短いほど返済額が増加 |

---

## キャッシュフローとデッドクロス

**デッドクロス**とは、ローンの元金返済額が減価償却費を上回る状態のことです。

- 減価償却費：帳簿上の経費（現金支出なし）
- 元金返済：現金支出あり（経費にならない）

デッドクロスが発生すると、帳簿上は利益が出ているのにキャッシュフローがマイナスになる場合があります。

---

## キャッシュフローの活用シーン

1. **購入判断** - 手残りがプラスになるかの確認
2. **借入条件の検討** - 金利・期間の影響をシミュレーション
3. **将来予測** - 空室率や金利変動の影響を把握

---

## キャッシュフローの注意点

- 減価償却費は現金支出を伴わない（帳簿上の経費）
- ローン元金返済は現金支出だが経費にならない
- 突発的な修繕費用は予測が難しい
