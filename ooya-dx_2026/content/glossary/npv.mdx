## NPV（正味現在価値）とは

**NPV（Net Present Value：正味現在価値）**は、将来得られるキャッシュフローを現在の価値に割り引いて合計し、初期投資額を差し引いた値です。

簡単に言うと、**「今の価値に換算して、いくら得するのか（損するのか）」**を表す指標です。

---

## なぜNPVが重要なのか

| 観点 | 説明 |
|------|------|
| 時間の価値 | 将来のお金を現在の価値に換算できる |
| 絶対額の把握 | 金額ベースで収益性を確認できる |
| 比較検討 | 異なる条件の案件を同じ基準で比較できる |

NPVがプラスであれば投じた資金以上のリターンが期待できることを意味し、マイナスであれば期待リターンを下回る可能性を示唆します。

---

## NPVの計算式

```
NPV = Σ (CFt / (1 + r)^t) - C0
```

- CFt：t年目のキャッシュフロー
- r：割引率
- t：年数
- C0：初期投資額

---

## NPVの計算例

| 項目 | 金額 |
|------|------|
| 初期投資額 | 1,000万円 |
| 1年目CF | 120万円 |
| 2年目CF | 120万円 |
| 3年目CF | 120万円 |
| 3年目売却額 | 1,050万円 |
| 割引率 | 5% |

```
NPV = 120/(1.05)^1 + 120/(1.05)^2 + (120+1050)/(1.05)^3 - 1000
    = 114.3 + 108.8 + 1010.5 - 1000
    = 約233.6万円
```

## NPVとIRRの違い

| 指標 | 表すもの | 単位 |
|------|---------|------|
| NPV | 収益の絶対額 | 円 |
| IRR | 収益率 | % |

NPVは「いくら儲かるか」、IRRは「何%のリターンか」を表します。両方を組み合わせて検討することで、より多角的な分析が可能です。

---

## NPVの活用シーン

1. **物件取得の検討** - 購入価格が妥当かどうかの参考になる
2. **売却タイミング** - 売却時期による収益性の変化を把握
3. **融資条件の比較** - 金利や借入期間の違いによる影響を確認

---

## NPVの注意点

- 割引率の設定によって結果が大きく変わる
- 将来のキャッシュフロー予測に依存する
- 売却価格の想定で結果が変動する
