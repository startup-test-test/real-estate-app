## 運営経費率とは

**運営経費率（OPEX Ratio）**は、家賃収入（またはEGI）に対する運営経費の割合を示す指標です。

物件の**運営効率**を測る目安として使われ、NOI（営業純収益）の計算に影響します。

---

## 運営経費率の計算式

```
運営経費率 = 運営経費 ÷ 家賃収入 × 100（%）
```

### 計算例

| 項目 | 金額 |
|------|------|
| 年間家賃収入 | 600万円 |
| 運営経費 | 120万円 |

```
運営経費率 = 120万円 ÷ 600万円 × 100 = 20%
```

---

## 運営経費に含まれる項目

| 経費項目 | 内容 |
|---------|------|
| 管理委託費 | 管理会社への委託料（賃料の3〜5%程度） |
| 修繕費 | 原状回復、小修繕など |
| 共用部光熱費 | 共用廊下、エレベーターなどの電気代 |
| 火災保険料 | 建物の保険料 |
| 固定資産税・都市計画税 | 土地・建物にかかる税金 |
| その他 | 清掃費、消防設備点検費など |

---

## 運営経費率の目安

| 物件タイプ | 運営経費率の目安 |
|-----------|----------------|
| 区分マンション | 15〜25%程度 |
| 一棟アパート | 15〜20%程度 |
| 一棟マンション | 20〜30%程度 |
| 一棟ビル | 25〜35%程度 |

エレベーターや共用設備が多い物件ほど、経費率は高くなる傾向があります。

---

## 運営経費率とNOIの関係

NOIは以下の式で計算されます：

```
NOI = 家賃収入 × (1 − 空室率) × (1 − 運営経費率)
```

簡易的な計算例（家賃収入600万円、空室率10%、運営経費率20%）：

```
NOI = 600万円 × 0.9 × 0.8 = 432万円
```

---

## 運営経費率が高いとどうなるか

- NOI（営業純収益）が低下
- 実質利回りが低下
- キャッシュフローが減少

---

## 運営経費率を抑える方法

1. **管理会社の見直し** - 管理委託費の交渉や変更
2. **共用部の省エネ化** - LED照明、人感センサーなど
3. **保険の見直し** - 適正な補償内容への見直し
4. **修繕計画の最適化** - 計画的な修繕で突発支出を抑制

---

## 注意点

- 運営経費には**ローン返済は含まない**
- 大規模修繕費は別途積立で考えることが多い
- 築年数が経つと修繕費が増加する傾向
