## EGI（有効総収入）とは

**EGI（Effective Gross Income：有効総収入）**は、GPI（総潜在収入）から**空室損や未回収損**を差し引いた、実質的に見込める収入です。

「実効総収入」とも呼ばれ、**NOI計算の基礎**となる重要な指標です。

---

## EGIの計算式

```
EGI = GPI − 空室損 − 未回収損
```

または

```
EGI = GPI × (1 − 空室率) × (1 − 未回収率)
```

---

## 計算例

| 項目 | 金額・率 |
|------|---------|
| GPI（総潜在収入） | 600万円 |
| 想定空室率 | 10% |
| 未回収率 | 1% |

```
EGI = 600万円 × (1 − 0.10) × (1 − 0.01)
    = 600万円 × 0.90 × 0.99
    = 534.6万円
```

---

## 空室損と未回収損

### 空室損

空室期間中に得られない賃料収入のこと。

| 要因 | 内容 |
|------|------|
| 退去後の空室期間 | 原状回復〜次の入居まで |
| 募集中の空室 | 入居者が見つからない期間 |

### 未回収損

入居中でも回収できない賃料のこと。

| 要因 | 内容 |
|------|------|
| 賃料滞納 | 入居者の支払い遅延・未払い |
| 貸倒れ | 回収不能となった賃料 |

---

## GPIからNOIまでの流れ

```
GPI（総潜在収入）
　↓ − 空室損・未回収損
EGI（有効総収入）
　↓ − 運営経費
NOI（営業純収益）
```

---

## EGIの目安

| 空室率 | GPI 600万円の場合のEGI |
|--------|----------------------|
| 5% | 約570万円 |
| 10% | 約540万円 |
| 15% | 約510万円 |

※未回収損を考慮しない簡易計算

---

## EGIを使う場面

1. **収支シミュレーション** - 現実的な収入見込みの算出
2. **NOI計算** - EGIから経費を引いてNOIを算出
3. **実質利回り計算** - より現実的な利回り計算

---

## 運営経費率との関係

運営経費率はEGIに対する割合で計算されることもあります：

```
NOI = EGI × (1 − 運営経費率)
```

例：EGI 540万円、運営経費率20%の場合

```
NOI = 540万円 × 0.8 = 432万円
```

---

## EGIの注意点

- 空室率・未回収率の想定によって大きく変わる
- 楽観的な想定は収支計画を狂わせる原因になる
- 金融機関は一定の空室率を見込んで審査する
