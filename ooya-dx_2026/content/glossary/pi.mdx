## PI（収益性指数）とは

**PI（Profitability Index：収益性指数）**は、投資額1円あたりどれだけの現在価値を生み出すかを表す指標です。

簡単に言うと、**「投資額に対して何倍の価値が得られるか」**を表します。

---

## なぜPIが重要なのか

| 観点 | 説明 |
|------|------|
| 投資効率の把握 | 限られた資金での効率を比較できる |
| 相対評価 | 規模の異なる案件を比較しやすい |
| 意思決定の補助 | 複数案件の優先順位付けに活用できる |

PIは特に資金制約がある場合に、どの案件に優先的に投資すべきかを検討する際の参考になります。

---

## PIの計算式

```
PI = 将来キャッシュフローの現在価値合計 ÷ 初期投資額
```

または、NPVを使って：

```
PI = (NPV + 初期投資額) ÷ 初期投資額
PI = 1 + (NPV ÷ 初期投資額)
```

---

## PIの計算例

| 項目 | 金額 |
|------|------|
| 初期投資額 | 1,000万円 |
| 将来CFの現在価値合計 | 1,200万円 |

```
PI = 1,200万円 ÷ 1,000万円 = 1.2
```

PI = 1.2 は、投資額1円あたり1.2円の価値が得られることを意味します。

---

## PIの判断基準

| PI値 | 意味 |
|------|------|
| PI > 1 | 投資額以上の価値が得られる可能性 |
| PI = 1 | 投資額と得られる価値が同等 |
| PI < 1 | 投資額を下回る可能性 |

一般的にPI > 1の案件が検討対象となりますが、PIの数値だけで判断せず、他の指標と合わせて総合的に検討することが重要です。

---

## PIとNPV・IRRの関係

| 指標 | 表すもの | 単位 | 特徴 |
|------|---------|------|------|
| NPV | 収益の絶対額 | 円 | 金額ベースで把握 |
| IRR | 収益率 | % | 年率で把握 |
| PI | 投資効率 | 倍 | 相対的な効率を把握 |

3つの指標を組み合わせることで、より多角的な分析が可能です。

---

## PIの活用シーン

1. **複数案件の比較** - 資金制約がある中での優先順位付け
2. **投資効率の確認** - 効率的な資金配分の検討
3. **代替案の評価** - 異なる投資オプションの比較

---

## PIの注意点

- 案件の規模（投資額の大きさ）は考慮されない
- 割引率の設定によって結果が変わる
- 将来のキャッシュフロー予測に依存する
