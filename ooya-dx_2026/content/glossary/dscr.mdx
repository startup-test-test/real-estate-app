## DSCR（債務返済カバー率）とは

**DSCR（Debt Service Coverage Ratio：債務返済カバー率）**は、物件から得られる年間の純営業収益（NOI）が、年間のローン返済額（ADS）の何倍あるかを示す指標です。

金融機関が融資審査で重視する指標の一つで、**返済余力**を測る目安として使われます。

---

## DSCRの計算式

```
DSCR = NOI（年間純営業収益） ÷ ADS（年間元利返済額）
```

### 計算例

| 項目 | 金額 |
|------|------|
| 年間家賃収入 | 600万円 |
| 運営経費 | 100万円 |
| NOI | 500万円 |
| 年間ローン返済額（ADS） | 400万円 |

```
DSCR = 500万円 ÷ 400万円 = 1.25
```

---

## DSCRの目安

| DSCR | 評価 |
|------|------|
| 1.0未満 | 返済不能（NOI＜返済額） |
| 1.0〜1.2 | 余裕がない |
| **1.2〜1.3** | **金融機関の最低基準（目安）** |
| 1.3以上 | 安定した返済余力 |

金融機関によって基準は異なりますが、**DSCR 1.2以上**を求められることが多いです。

---

## DSCRが低いとどうなるか

- 空室や修繕で収入が減ると、すぐに返済が厳しくなる
- 金融機関からの融資が受けにくくなる
- 金利上昇時のリスクが高い

---

## DSCRを改善する方法

1. **収入を増やす** - 賃料の見直し、空室対策
2. **経費を減らす** - 管理費・修繕費の見直し
3. **返済額を減らす** - 借入期間の延長、金利交渉
