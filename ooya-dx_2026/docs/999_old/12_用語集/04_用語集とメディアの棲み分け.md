# コンテンツ戦略：用語集とメディアの役割分担

## 概要

用語集（/glossary/）とメディア（/media/）の役割を明確に分けることで、SEO効果を最大化し、重複コンテンツを回避します。

---

## 用語集（/glossary/）の役割

### 掲載するコンテンツ

- **「〇〇とは？」の基本定義**
- 計算式・計算方法
- 計算例（具体的な数値例）
- 目安・基準値・評価表
- 関連用語との違い（比較表）
- 関連シミュレーターへのリンク

### 例

| ページ | 内容 |
|--------|------|
| `/glossary/irr` | IRRの定義、計算式、目安、CCRとの違い |
| `/glossary/noi` | NOIの定義、計算式、NCFとの違い |
| `/glossary/ccr` | CCRの定義、計算式、レバレッジ効果の説明 |

### 特徴

- **辞書的な内容**：簡潔で正確な定義
- **計算重視**：式と具体例を必ず含める
- **ツール連携**：関連シミュレーターへ誘導

---

## メディア（/media/）の役割

### 掲載するコンテンツ

- 実践的な活用法・応用テクニック
- よくある失敗と対策
- 具体的なケーススタディ・事例
- 複数の概念を組み合わせた解説
- 初心者向けの入門ガイド
- 比較記事・選び方ガイド

### 例

| ページ | 内容 |
|--------|------|
| `/media/irr-vs-ccr-comparison` | IRRとCCR、どちらを重視すべき？ |
| `/media/irr-improvement-tips` | IRRを高める5つの方法 |
| `/media/noi-calculation-mistakes` | NOI計算でよくある3つの間違い |
| `/media/beginner-investment-metrics` | 初心者が覚えるべき投資指標5選 |
| `/media/cash-flow-analysis-case-study` | 【実例】築30年アパートの収益分析 |

### 特徴

- **実践的な内容**：読者が行動に移せる情報
- **ストーリー性**：事例やケーススタディを含む
- **深掘り**：1つのテーマを詳しく解説

---

## 相互リンクの構造

```
┌─────────────────────────────────────────────────────────┐
│                      用語集                              │
│  /glossary/irr                                          │
│  ├─ IRRの定義・計算式・目安                              │
│  ├─ 関連ツール → /simulator                             │
│  └─ 関連記事 → /media/irr-improvement-tips             │
└─────────────────────────────────────────────────────────┘
                           ↑↓
┌─────────────────────────────────────────────────────────┐
│                      メディア                            │
│  /media/irr-improvement-tips                            │
│  ├─ IRRを高める具体的な方法                              │
│  ├─ 用語解説 → /glossary/irr                            │
│  └─ 計算ツール → /simulator                             │
└─────────────────────────────────────────────────────────┘
```

### リンクのルール

1. **メディア記事内**で専門用語が出てきたら、用語集へリンク
2. **用語集**から関連する実践記事があればメディアへリンク
3. **両方**から関連シミュレーターへリンク

## SEOキーワードの棲み分け

### 用語集で狙うキーワード

- 「〇〇とは」
- 「〇〇 意味」
- 「〇〇 計算方法」
- 「〇〇 計算式」
- 「〇〇 目安」

### メディアで狙うキーワード

- 「〇〇 コツ」
- 「〇〇 方法」
- 「〇〇 失敗」
- 「〇〇 事例」
- 「〇〇 比較」
- 「〇〇 おすすめ」
- 「〇〇 初心者」

---

## 重複を避けるためのチェックリスト

記事を書く前に確認：

- [ ] 「〇〇とは？」の定義が主な内容 → **用語集**へ
- [ ] 実践的なノウハウが主な内容 → **メディア**へ
- [ ] 同じキーワードで用語集に既にページがある → メディアでは別の切り口で書く
- [ ] 用語集のページからメディア記事へリンクを追加したか
- [ ] メディア記事から用語集へリンクを追加したか

## 具体的な運用例

### 例：「キャップレート」について書く場合

#### 用語集に書く内容（/glossary/cap-rate）

```markdown
## キャップレートとは
キャップレート（Capitalization Rate）は、不動産の収益性を...

## 計算式
キャップレート = NOI ÷ 物件価格 × 100

## 計算例
| 項目 | 金額 |
|------|------|
| NOI | 500万円 |
| 物件価格 | 1億円 |
| キャップレート | 5% |

## 目安
| キャップレート | 評価 |
|---------------|------|
| 3%未満 | 都心一等地 |
| 3〜5% | 都市部標準 |
...
```

#### メディアに書く内容（/media/cap-rate-negotiation）

```markdown
## キャップレートを使った価格交渉術

物件を購入する際、売主の提示価格が適正かどうか
判断するのは難しいですよね。

そこで活用したいのがキャップレートです。
※キャップレートの基本については[用語集](/glossary/cap-rate)を参照

### 実際の交渉事例

先日、築15年のRCマンションを検討したお客様の事例です...
```

## まとめ

| 項目 | 用語集 | メディア |
|------|--------|----------|
| 目的 | 定義・計算方法の解説 | 実践ノウハウの提供 |
| 文体 | 辞書的・簡潔 | ストーリー性・読み物 |
| 長さ | 1,000〜2,000字 | 2,000〜5,000字 |
| 更新頻度 | 低（定義は変わらない） | 高（事例・トレンド更新） |
| SEOキーワード | 「〇〇とは」「計算」 | 「コツ」「方法」「事例」 |
