## リフォーム会社との提携

### 提携先の条件

| 条件 | 理由 |
|------|------|
| 収益物件の実績あり | 大家さんのニーズを理解 |
| 原状回復対応可 | 最も頻度が高い |
| 見積もり無料 | ユーザーハードルを下げる |
| 対応エリア | 最初は首都圏から |

### 提携の進め方

```
【Step 1】地元の工務店1〜2社と関係構築
    ↓
【Step 2】実際にマッチングしてみる（テスト）
    ↓
【Step 3】手数料・契約条件を決定
    ↓
【Step 4】掲載会社を拡大
```

## 競合分析

### 既存のリフォームマッチングサービス

| サービス | ターゲット | 強み | 弱み |
|----------|-----------|------|------|
| くらしのマーケット | 一般消費者 | 知名度、掲載数 | 大家向けではない |
| ホームプロ | 一般消費者 | 大手、信頼性 | 収益物件に不向き |
| リショップナビ | 一般消費者 | 比較機能 | ROI試算なし |

### 大家DXの差別化

```
【競合】一般向けリフォームマッチング
         ↓
    見積もり比較で終わり

【大家DX】収益物件オーナー専用
         ↓
    見積もり + ROI試算 + 入居率改善事例
         ↓
    「投資判断」ができる
```

## リスクと対策

| リスク | 対策 |
|--------|------|
| ユーザーが集まらない | SEO/コンテンツに注力、広告検討 |
| リフォーム会社との提携が進まない | 地元の工務店から小さく始める |
| 有料転換率が低い | 無料で価値を体験させ、信頼を構築してから |
| 競合参入 | 「大家さん専用」のポジションを確立 |
