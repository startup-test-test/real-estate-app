# 独自性と実体験 コンテンツ戦略

## 目的

- GoogleのE-E-A-T（Experience）で競合と差別化
- 企業メディア（suumo, homes等）には書けない個人の実体験を武器にする
- 具体的な金額・交渉プロセスのリアルな情報で読者の信頼を獲得

---

## 筆者の不動産取引実績

### 取引①：標準的な仲介手数料（3%）

| 項目 | 内容 |
|------|------|
| 物件価格 | 6,980万円 |
| 仲介手数料率 | 3% + 6万円（税別） |
| 手数料額 | 約215.4万円（税別） |
| 取引種別 | 売買 |
| ポイント | 上限いっぱいの標準的な手数料。計算ツールとの比較素材 |

### 取引②：売主直売で仲介手数料ゼロ

| 項目 | 内容 |
|------|------|
| 物件価格 | 5,530万円 |
| 仲介手数料率 | **0%（売主直売）** |
| 手数料額 | **0円** |
| 取引種別 | 新築購入 |
| ポイント | 売主から直接購入したため仲介手数料が不要。「無料のからくり」ページの実例 |

### 取引③：交渉で仲介手数料を半額に

| 項目 | 内容 |
|------|------|
| 物件価格 | 8,100万円 |
| 仲介手数料率 | **1.5%（上限の半額）** |
| 手数料額 | 約121.5万円（税別） |
| 取引種別 | 売買 |
| ポイント | 交渉により半額を実現。交渉ガイドページの核となる体験談 |

### 取引④：仲介手数料のオーバーローン組み込み

| 項目 | 内容 |
|------|------|
| 対象取引 | （該当する取引を記載） |
| 住宅ローン | 物件価格 + 仲介手数料をオーバーローンで組み込み |
| ポイント | 仲介手数料を現金で用意せず住宅ローンに含めた実例。支払い方法・ローンページの独自コンテンツ |
| 書ける内容 | 銀行との交渉過程、審査のポイント、金利への影響、メリット・デメリット |

### 取引⑤：賃貸の仲介手数料を半額にした体験

| 項目 | 内容 |
|------|------|
| 取引種別 | 賃貸契約 |
| 仲介手数料率 | **0.5ヶ月分（半額）** |
| ポイント | 賃貸でも仲介手数料の交渉で半額を実現 |
| 注意点 | 仲介手数料を半額にした代わりに、アルコール消毒（室内消毒費）等の名目で別途費用を請求された |
| 書ける内容 | 半額交渉の方法、消毒費・除菌費等の「隠れコスト」の実態、トータルで本当に安くなったか検証、賃貸初期費用の内訳チェックリスト |

### 取引⑥：築古・低額物件（300万円）の仲介手数料

| 項目 | 内容 |
|------|------|
| 物件価格 | 300万円 |
| 取引種別 | 売買（築古物件） |
| 仲介手数料 | 法改正前の計算方式で算出 |
| ポイント | 低額物件は手数料率が高くなる（200万円以下：5%、200〜400万円：4%）。2024年の法改正で低額物件の仲介手数料上限が変更されたが、この取引は改正前 |
| 書ける内容 | 低額物件の手数料計算の実例、改正前後の比較、築古物件購入の注意点 |

### 実務体験⑦：仲介手数料の勘定科目・経費処理

| 項目 | 内容 |
|------|------|
| 体験内容 | 不動産取引で支払った仲介手数料の会計処理を実際に行った |
| ポイント | 自身の実体験として「こう処理した」事実を書ける |
| 注意事項 | **「こうすべき」「こうやるべき」等の助言は書かない**（非弁行為・税理士法抵触リスク回避）。あくまで「筆者はこう処理した」という体験の記述に留める |
| 書ける内容 | 実際の仕訳の記録、使用した勘定科目、確定申告での扱い（全て自身の体験として） |
| 書かないこと | 読者への具体的な税務アドバイス、「〜すべき」「〜が正しい」等の断定的表現 |
| 推奨表現 | 「筆者の場合は〜で処理しました」「詳しくは税理士にご相談ください」 |

## 各ページの体験コンテンツ案

### 計算ツールページ（取引①）

> **H2: 筆者の実例｜6,980万円の売買で仲介手数料はいくらだった？**
>
> - 計算ツールで6,980万円を入力 → 結果：約215.4万円
> - 実際に支払った金額と一致するか比較
> - 「計算通りの金額を請求されました」というリアルな裏付け
> - 読者が自分の物件価格で計算する動機づけ

### 交渉ガイドページ（取引③）

> **H2: 筆者の実例｜8,100万円の物件で仲介手数料を半額にした方法**
>
> - 交渉前：上限3%で約249万円
> - 交渉後：1.5%で約121.5万円 → **約127万円の節約**
> - 交渉の切り出し方、タイミング、不動産会社の反応
> - 成功した理由の分析

### 無料・安いページ（取引②）

> **H2: 筆者の実例｜5,530万円の新築を仲介手数料ゼロで購入**
>
> - 売主（デベロッパー）から直接購入
> - 仲介会社を通さないメリット・デメリット
> - 本来かかるはずだった手数料：約178万円が丸々不要に
> - 売主直売物件の見つけ方

### 支払い方法・ローンページ（取引④）

> **H2: 筆者の実例｜仲介手数料を住宅ローンに組み込んだ話**
>
> - オーバーローンで仲介手数料を住宅ローンに含めた経緯
> - 銀行への相談・審査プロセス
> - 手元資金を温存できたメリット
> - 金利負担の実際のコスト（長期で見た総支払額）
> - 注意点：全ての銀行で可能ではない

### 賃貸の仲介手数料ページ（取引⑤）

> **H2: 筆者の実例｜賃貸の仲介手数料を半額にしたら「消毒費」を請求された話**
>
> - 仲介手数料を0.5ヶ月分に交渉成功
> - しかしアルコール消毒（室内消毒費）の名目で別途費用が発生
> - 仲介手数料を安くしても初期費用のトータルは変わらない場合がある
> - 賃貸の初期費用で注意すべき「隠れコスト」一覧
>   - 消毒費／除菌費／害虫駆除費
>   - 鍵交換費用
>   - 安心サポート費 等
> - 交渉する際はトータルコストで比較すべき

## メモ

- 取引④（オーバーローン）の詳細（物件価格・銀行名等）は後日追記
- 写真・書類の一部（個人情報除く）を掲載できればさらに信頼性UP
- 各体験談は「結論→詳細→読者へのアドバイス」の構成で統一する
