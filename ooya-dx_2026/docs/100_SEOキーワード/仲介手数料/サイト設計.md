# 仲介手数料 サイト設計（v2 網羅版）

## 概要

仲介手数料の計算ツール（既存）を軸に、トピカルオーソリティを構築するコンテンツクラスタ設計。
13ページで仲介手数料トピックを網羅し、115,420 vol/月をカバーする。
**1つのKWは1ページのみに割り当て、カニバリを防止する。**

## カニバリ防止ルール

複合KWの割り当ては**ユーザーが誰か（主意図）**で決める。

| KW例 | 主意図 | 割り当て先 |
|------|--------|-----------|
| 仲介手数料 不動産 計算 | 計算したい | 計算ツール |
| 仲介手数料 不動産 安い | 安くしたい | 無料の仕組み |
| 仲介手数料 不動産 賃貸 | 賃貸の人 | 賃貸ガイド |
| 仲介手数料 相場 賃貸 | 賃貸の人 | 賃貸ガイド |
| 仲介手数料 上限 賃貸 | 賃貸の人 | 賃貸ガイド |
| 仲介手数料無料 賃貸 | 賃貸の人 | 賃貸ガイド |
| 仲介手数料 不動産 消費税 | 税金が気になる | 消費税 |

ピラーページは**浅く広く概要のみ**。各H2は2〜3行 + 「詳しくはこちら→」リンク。
