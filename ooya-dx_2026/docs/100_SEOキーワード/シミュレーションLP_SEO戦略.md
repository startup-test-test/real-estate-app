## サイト全体のKW棲み分け

### ページ階層とメインKW

```
/tools（ピラーページ）──── KW:「不動産投資 計算ツール」「不動産投資 計算」
  │
  ├── /tools/simulator（LP）── KW:「不動産投資 シミュレーション」  ← CV地点
  ├── /tools/yield-rate ────── KW:「利回り 計算」
  ├── /tools/brokerage ─────── KW:「仲介手数料 計算」
  ├── /tools/depreciation ──── KW:「減価償却 計算」
  ├── /tools/mortgage-loan ─── KW:「ローン 返済 計算」
  └── ... 他22ツール

/media（記事）
  ├── /media/simulation/excel-simulation ──── KW:「シミュレーション エクセル 無料」
  ├── /media/simulation/depreciation ──────── KW:「減価償却 シミュレーション」
  ├── /media/brokerage/... ────────────────── KW:「仲介手数料」系
  └── ...
```

### KWカニバリ防止ルール

| ページ種別 | メインKW | 役割 |
|---|---|---|
| /tools | **計算ツール** / **計算** | ツール一覧ハブ（各ツールへの導線） |
| /tools/simulator | **シミュレーション** | 総合シミュレーションLP（CV地点） |
| /tools/[個別ツール] | **[指標名] 計算** | 単体計算ツール |
| /media/[記事] | **[テーマ] シミュレーション** or 情報系KW | 情報提供 → ツール/LPへ誘導 |

- /tools には「シミュレーション」を入れない（「計算ツール」に集中）
- /tools/simulator には「計算ツール」を入れない（「シミュレーション」に集中）
- 記事は情報系KWで集客し、CTA経由でツールページ or LPへ送る

### 内部リンク設計

```
            記事（情報系KW）
              │ CTA
              ▼
/tools ◄──► /tools/simulator（LP）◄── 検索「シミュレーション」
  │              ▲
  │              │ CTA「総合シミュレーションを試す」
  ▼              │
/tools/[個別ツール] ──┘
  │
  └──► /media/[関連記事] へ相互リンク
        例: /tools/depreciation ◄──► /media/simulation/depreciation
        例: /tools/brokerage ◄──► /media/brokerage/...
```

各ツールページからの導線:
- → /tools（パンくず・ピラー強化）
- → /tools/simulator（CTA「総合シミュレーションを試す」）
- → 関連記事（「詳しく知りたい方はこちら」）
