## 概要

大家DXサイト内で提供する「賃貸経営用語集」の管理ドキュメントです。

- **URL**: https://ooya.tech/glossary

## 運用方針

### コンテンツ方針

1. **シミュレーターとの連携**
   - 用語解説→関連シミュレーターへの導線を設ける
   - 「実際に計算してみる」CTAを設置

2. **SEO対策**
   - タイトル: 「〇〇とは？不動産での計算方法と目安を解説」形式
   - 構造化データ: FAQスキーマを自動生成
   - 内部リンク: 関連用語への相互リンク

3. **品質基準**
   - 正確な情報（税制改正等は最新化）
   - 計算例・具体例を含める
   - 初心者にもわかりやすい表現

### カテゴリ分類

| カテゴリ | 内容 |
|---------|------|
| 投資指標 | IRR, CCR, ROI, NPV など |
| 収益指標 | NOI, NCF, 利回り など |
| 融資関連 | LTV, DSCR, K% など |
| 税金関連 | 減価償却, 譲渡所得税 など |
| 評価手法 | 積算評価, 収益還元法 など |
