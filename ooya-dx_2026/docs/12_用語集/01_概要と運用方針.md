## 運用ワークフロー

### シミュレーター追加時の用語管理

新しいシミュレーターを作成した際は、以下のワークフローで用語を管理します。

```
新シミュレーター作成
    ↓
使用する専門用語を特定
    ↓
05_用語集マスターデータ.md に追加
    ↓
優先度に応じて用語集ページ作成
```

### マスターデータの役割

| 役割 | 内容 |
|-----|------|
| 用語の一元管理 | 全用語をナンバリングで管理 |
| ツール連携 | どのシミュレーターでどの用語を使うか把握 |
| 進捗管理 | 公開状況・チェック状況を追跡 |
| 優先度判断 | 次に作成すべき用語を決定 |

### 用語追加時のルール

- 新シミュレーター作成時に関連用語をマスターデータに追加
- 「関連ツール」列に該当シミュレーターのパスを記載
- 用語名に説明がない場合は括弧で日本語説明を補足
  - 例: `DCF法` → `DCF法（割引キャッシュフロー法）`

## 運用方針

### コンテンツ方針

1. **シミュレーターとの連携**
   - 用語解説→関連シミュレーターへの導線を設ける
   - 「実際に計算してみる」CTAを設置

2. **SEO対策**
   - タイトル: 「〇〇とは？不動産での計算方法と目安を解説」形式
   - 構造化データ: FAQスキーマを自動生成
   - 内部リンク: 関連用語への相互リンク

3. **品質基準**
   - 正確な情報（税制改正等は最新化）
   - 計算例・具体例を含める
   - 初心者にもわかりやすい表現

### カテゴリ分類

| カテゴリ | 内容 |
|---------|------|
| 収益指標 | IRR, CCR, NOI, NCF, 利回り など |
| 融資関連 | LTV, DSCR, K% など |
| 税金関連 | 減価償却, 譲渡所得税 など |
| 評価手法 | 積算評価, 収益還元法, DCF法 など |
| 売却関連 | 仲介手数料, 売却諸経費 など |
| 物件・建物関連 | 路線価, 築年数, 建物構造 など |
