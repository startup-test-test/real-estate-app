# 用語作成手順書

新しい用語を作成する際の具体的な手順です。

---

## Step 1: 用語の調査

### 1-1. SEOキーワード調査

以下のツールでキーワードを調査します。

- **Ubersuggest**: https://neilpatel.com/jp/ubersuggest/
- **ラッコキーワード**: https://related-keywords.com/
- **Google キーワードプランナー**

確認項目:
- [ ] 月間検索ボリューム
- [ ] 競合性（難易度）
- [ ] 関連キーワード

### 1-2. 競合調査

検索上位10サイトを確認:
- [ ] どのような構成か
- [ ] 何が書かれているか
- [ ] 差別化ポイントは何か

### 1-3. 内容の正確性確認

- [ ] 国税庁・金融庁等の公式情報
- [ ] 専門書籍・論文
- [ ] 税制改正の最新情報

---

## Step 2: 執筆準備

### 2-1. アウトライン作成

```
1. 〇〇とは（定義）
2. なぜ重要か（意義）
3. 計算式・計算方法
4. 具体的な計算例
5. 計算結果の見方（※評価表現は禁止）
6. 活用シーン
7. 注意点・よくある誤解
8. 関連用語との違い
9. シミュレーターへの誘導
10. 免責事項
```

### 2-2. 必要素材の準備

- [ ] 計算式
- [ ] 計算例の数値
- [ ] 目安となる基準値
- [ ] 図表（必要に応じて）

---

## Step 3: MDXファイル作成

### 3-1. ファイル作成

```bash
# content/glossary/ に新規ファイルを作成
touch content/glossary/{slug}.mdx
```

### 3-2. フロントマター設定

```mdx
---
title: "〇〇（読み方）とは？不動産での計算方法と目安をわかりやすく解説"
shortTitle: "〇〇（読み方）"
description: "〇〇は、△△を表す指標です。不動産における〇〇の計算方法、目安となる数値、活用方法を初心者にもわかりやすく解説します。"
category: "収益指標"
relatedTools: "/simulator"
keywords: ["〇〇 とは", "〇〇 計算", "〇〇 不動産", "〇〇 目安"]
---
```

**title のコツ:**
- 「〇〇とは？」で始める
- 「計算方法」「目安」など具体的なベネフィットを含める
- 30〜60文字程度

**description のコツ:**
- 120文字程度
- キーワードを自然に含める
- 読者が得られる情報を明示

### 3-3. 本文執筆

#### 導入部（〇〇とは）

```mdx
## 〇〇とは

**〇〇（英語名：English Name）**は、△△を表す指標です。

簡単に言うと、**「□□□□」**を表します。
```

ポイント:
- 最初の1文で定義を明確に
- 「簡単に言うと」で初心者向けの説明を追加

#### 計算式セクション

```mdx
## 〇〇の計算式

\`\`\`
〇〇 = A ÷ B × 100（%）
\`\`\`

### 計算例

| 項目 | 金額 |
|------|------|
| A | 1,000万円 |
| B | 100万円 |

\`\`\`
〇〇 = 100万円 ÷ 1,000万円 × 100 = 10%
\`\`\`
```

ポイント:
- 計算式はコードブロックで見やすく
- 具体的な数値例を必ず含める

#### 目安セクション

⚠️ **法務上の注意**: 評価表現（「良好」「優良」「標準的」等）を含む目安表は作成しないでください。投資判断の助言と見なされる可能性があります。

```mdx
## 〇〇の計算結果の見方

〇〇は物件タイプ、エリア、築年数などによって大きく異なります。
同条件の物件と比較検討する際の参考値としてご活用ください。

※本計算結果は参考情報です。実際の経営判断は専門家にご相談ください。
```

ポイント:
- 数値の「評価」「判定」は行わない
- 比較検討の参考値として提示
- 必ず免責事項を含める

#### CTA（シミュレーター誘導）

```mdx
## 実際に〇〇を計算してみる

大家DXのシミュレーターでは、物件情報を入力するだけで〇〇を自動計算できます。

[シミュレーターで〇〇を計算する →](/simulator)
```

## よくある質問

### Q: カテゴリはどう決める？

既存のカテゴリに合わせてください。新規カテゴリが必要な場合は相談。

| カテゴリ | 含まれる用語例 |
|---------|--------------|
| 収益指標 | IRR, CCR, NOI, NCF, 利回り |
| 融資関連 | LTV, DSCR, K% |
| 税金関連 | 減価償却, 譲渡所得税 |
| 評価手法 | 積算評価, 収益還元法, DCF法 |
| 売却関連 | 仲介手数料, 譲渡所得税 |
| 物件・建物関連 | 路線価, 築年数, 建物構造 |

### Q: 関連ツールがない場合は？

`relatedTools` は省略可能です。関連シミュレーターがない場合は記載しなくてOK。

### Q: 画像を使いたい場合は？

`/public/images/glossary/` に配置し、MDX内で参照:

```mdx
![画像の説明](/images/glossary/example.png)
```
