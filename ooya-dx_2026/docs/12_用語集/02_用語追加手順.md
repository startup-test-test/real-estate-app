# 用語追加手順

## 新しい用語を追加する方法

### 1. MDXファイルを作成

`content/glossary/` に新しいMDXファイルを作成します。

**ファイル名**: `{slug}.mdx`（例: `dscr.mdx`）

---

### 2. テンプレートをコピー

以下のテンプレートをコピーして内容を編集してください。

```mdx
---
title: "〇〇（読み方）とは？不動産での計算方法と目安をわかりやすく解説"
shortTitle: "〇〇（読み方）"
description: "〇〇は、△△を表す指標です。不動産における〇〇の計算方法、目安となる数値、活用方法を初心者にもわかりやすく解説します。"
category: "投資指標"
relatedTools: "/simulator"
keywords: ["〇〇 とは", "〇〇 計算", "〇〇 不動産", "〇〇 目安"]
---

## 〇〇とは

**〇〇（英語名：English Name）**は、△△を表す指標です。

簡単に言うと、**「□□□□」**を表します。

---
```

---

### 3. フロントマター設定

| 項目 | 必須 | 説明 |
|------|------|------|
| `title` | ◯ | SEO用のタイトル（「〇〇とは？」形式推奨） |
| `shortTitle` | ◯ | 一覧表示用の短いタイトル |
| `description` | ◯ | メタディスクリプション（120文字程度） |
| `category` | ◯ | カテゴリ名（既存カテゴリと合わせる） |
| `relatedTools` | △ | 関連シミュレーターへのパス |
| `keywords` | △ | SEOキーワード配列 |

---

### 4. カテゴリ一覧

既存のカテゴリと合わせてください。

| カテゴリ名 | 含まれる用語例 |
|-----------|--------------|
| 投資指標 | IRR, CCR, ROI, NPV |
| 収益指標 | NOI, NCF, 利回り |
| 融資関連 | LTV, DSCR, K% |
| 税金関連 | 減価償却, 譲渡所得税 |
| 評価手法 | 積算評価, 収益還元法 |

---

### 5. 確認・公開

1. 開発サーバーで確認
   ```bash
   npm run dev
   # http://localhost:3002/glossary/{slug} で確認
   ```

2. 確認項目
   - [ ] タイトル・説明文が正しく表示される
   - [ ] カテゴリが正しく分類される
   - [ ] 関連ツールへのリンクが機能する
   - [ ] 関連用語が表示される

3. 本番反映
   - developブランチにpush → mainにマージで自動デプロイ

---

## 注意事項

- ファイル名（slug）は英数字・ハイフンのみ使用
- 画像を使う場合は `/public/images/glossary/` に配置
- 税制・法律に関する内容は最新情報を確認
