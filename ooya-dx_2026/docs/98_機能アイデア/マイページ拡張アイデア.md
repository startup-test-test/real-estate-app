# マイページ拡張アイデア

作成日: 2026-01-30

## 戦略

- **無料で提供** → ユーザー数を増やす
- **ナーチャリング** → 別サービスでマネタイズ
- **個人情報は持たない** → リスク回避

## 追加機能アイデア

### 優先度：高

#### 1. 所有物件 / 検討中の分類機能

**概要：**
シミュレーション保存時に「検討中」「所有物件」を選択できるようにする。

**メリット：**
- ユーザーの負担が少ない（入力1回）
- 保有資産の全体像が見える
- マイページに戻る理由ができる
- freeeと競合しない

**実装イメージ：**
```
シミュレーション保存時
  ↓
[検討中] or [所有物件] を選択
  ↓
ダッシュボードで自動集計
```

#### 2. ダッシュボード強化（予測値ベース）

**表示内容：**
- 所有物件数
- 総資産額
- 月間CF合計（予測）
- 年間CF合計（予測）

**イメージ：**
```
┌─────────────────────────┐
│ 所有物件: 3件           │
│ 総資産: 1.5億円         │
│ 月間CF合計: 42万円      │
│ 年間CF合計: 504万円     │
└─────────────────────────┘
```

**注意：**
- 月次の実績入力は不要（予測値のみ）
- freee/MFクラウドと競合しない領域

#### 3. 物件比較ツール

**概要：**
保存済みシミュレーションを2〜3件並べて比較。

**比較項目：**
- 利回り
- 月間CF
- 年間CF
- 借入残高推移

---

### 優先度：中

#### 4. レントロール作成（サーバー保存なし）

**概要：**
買い付け申込書と同じ方式。ブラウザで生成→PDF出力。

**ポイント：**
- サーバーに個人情報を保存しない
- 無料提供
- テンプレート提供のみ（記載アドバイスなし）

#### 5. 融資資料作成（サーバー保存なし）

**概要：**
銀行提出用の物件概要書をPDF生成。

#### 6. 物件URL自動入力

**概要：**
楽待・健美家のURLを入力すると、物件情報を自動取得してシミュレーションに反映。

---

### 優先度：低（将来）

#### 7. 目標設定・進捗トラッキング

- 「月CF100万円」等の目標設定
- 現在の所有物件からの進捗表示

#### 8. メール通知

- 金利変動アラート
- 確定申告時期リマインダー

#### 9. PDFエクスポート

- シミュレーション結果をPDF出力
- 銀行提出用フォーマット

## 法的確認事項（弁護士確認済み）

**買い付け申込書について：**
- サーバー通過なし → OK
- 無料提供 → OK
- 記載内容のアドバイスなし → OK
- 有料 + アドバイスあり → 行政書士法リスク

**注意文言：**
「売買契約を成立させるものではありません」と記載
