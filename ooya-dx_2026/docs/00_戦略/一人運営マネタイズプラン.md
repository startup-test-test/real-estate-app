# 一人運営マネタイズプラン

## 1. 基本方針

| 項目 | 内容 |
|------|------|
| 運営体制 | 一人運営 |
| 月収目標 | 10〜100万円 |
| 重視すること | **ストック型**（積み上がる収益） |
| 避けること | フロー型（都度の労働が必要） |

---

## 2. ストック型 vs フロー型

| 種類 | 特徴 | 例 |
|------|------|-----|
| **ストック型** | 一度獲得すれば継続収入 | 月額掲載料、広告枠 |
| フロー型 | 都度の労働で収入 | HP制作、コンサル |

### 一人運営ではストック型を優先すべき理由

| 理由 | 説明 |
|------|------|
| 時間の自由 | 作業しなくても収入が入る |
| 収益の安定 | 毎月の収入が読める |
| スケーラビリティ | 顧客が増えれば収入も増える |
| 体調リスク回避 | 倒れても収入が止まらない |

## 8. 優先順位

| 順位 | マネタイズ | 理由 |
|------|-----------|------|
| 1 | **リフォーム会社掲載** | ストック型、スケーラブル |

## 9. 成功の鍵

| ポイント | 説明 |
|----------|------|
| **大家会員数が最優先** | 全てのマネタイズの基盤 |
| **SEOで集客を自動化** | 広告費ゼロで会員獲得 |
| **低単価で数を取る** | 5,000円 × 100社 = 50万円 |
| **自動化できる仕組み** | 申込・請求・掲載を自動化 |
| **フロー型は避ける** | HP制作は余裕があるときだけ |

## 11. まとめ

### 一人運営でも月100万円は可能

| 収益源 | 月額 |
|--------|------|
| リフォーム会社掲載（100社） | 50万円 |
| 広告枠（5社） | 25万円 |
| アフィリエイト | 20万円 |
| メルマガ広告 | 5万円 |
| **合計** | **100万円** |

### ストック型のメリット

| メリット | 説明 |
|----------|------|
| 積み上がる | 解約がなければ増え続ける |
| 安定する | 毎月の収入が読める |
| 自由になる | 作業しなくても収入が入る |
| スケールする | 上限なく伸ばせる |
