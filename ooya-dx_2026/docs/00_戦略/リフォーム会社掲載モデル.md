# リフォーム会社掲載モデル

## 1. ビジネスモデル概要

| 項目 | 内容 |
|------|------|
| サービス | リフォーム会社の掲載・紹介 |
| ターゲット | 一人親方・小規模リフォーム会社 |
| 月額料金 | **5,000円（税別）** |
| 目標社数 | 100社 |
| 月間収益目標 | **50万円** |

---

## 2. なぜ月額5,000円なのか

### 競合との比較

| サービス | 料金体系 | ターゲット |
|----------|----------|-----------|
| ホームプロ | 成約報酬型（高額） | 中〜大規模 |
| リショップナビ | 成約報酬型 | 中規模以上 |
| くらしのマーケット | 手数料20% | 個人〜小規模 |
| **御社** | **月額5,000円固定** | **一人親方・小規模** |

### 5,000円の強み

| ポイント | 説明 |
|----------|------|
| 参入しやすい | 一人親方でも払える金額 |
| 解約されにくい | 「まぁいいか」で継続される |
| 営業しやすい | 「月5,000円で大家さんに露出」 |
| 成約報酬なし | リフォーム会社のリスクゼロ |
| 競合と差別化 | 大手は高額、御社は低価格固定 |

---

## 3. 大家さん案件の魅力

### リフォーム会社にとって大家さんが理想の顧客である理由

| 項目 | 一般客（自宅リフォーム） | 大家さん |
|------|------------------------|----------|
| 発注頻度 | 数年に1回 | **年に数回** |
| 物件数 | 1件 | **複数件** |
| 単価 | 高い（こだわり強い） | 適正（コスト意識） |
| 決裁スピード | 遅い（家族会議） | **早い** |
| リピート | ほぼなし | **高い** |
| 紹介 | 少ない | **大家コミュニティで広がる** |

### 一人親方が大家案件を欲しがる理由

| 理由 | 説明 |
|------|------|
| 大手に取られない | 大家さんは「安くて早い」を好む |
| 営業コストが低い | 1人捕まえれば継続発注 |
| 柔軟対応できる | 小回りが利く |
| ホームプロ等が高い | 手数料払えない、御社なら払える |

## 8. 集客方法（リフォーム会社の獲得）

### 獲得チャネル

| 方法 | 説明 | コスト |
|------|------|--------|
| サイト内募集 | 「掲載しませんか？」ページ | 無料 |
| 大家さん経由 | 「おすすめ会社を教えて」で獲得 | 無料 |
| Google検索 | 地域のリフォーム会社にメール | 無料 |
| SNS | X（Twitter）で告知 | 無料 |
| チラシ | 建材店・ホームセンターに設置 | 少額 |

### 営業トーク例

```
「大家さん向けの不動産投資サイトを運営しています。
会員数は〇〇名で、リフォーム会社を探している方が多いです。
月額5,000円でサイトに掲載できます。
成約報酬は一切ありません。興味ありませんか？」
```

## 9. 運用の手間

### 一人運営で可能か

| 作業 | 頻度 | 時間 |
|------|------|------|
| 新規掲載対応 | 週1〜2件 | 30分/件 |
| 問い合わせ対応 | 月数件 | 10分/件 |
| 請求・入金確認 | 月1回 | 1時間 |
| 掲載情報更新 | 随時 | 対応不要（会社が直接編集） |

**月間作業時間：5〜10時間程度**

### 自動化できる部分

| 作業 | 自動化方法 |
|------|-----------|
| 申込受付 | Webフォーム |
| 請求 | Stripe等の定期課金 |
| 掲載情報入力 | 会社側が直接入力 |
| 解約 | 自動処理 |

## 10. リスクと対策

| リスク | 対策 |
|--------|------|
| 掲載会社が集まらない | 最初は無料キャンペーンで獲得 |
| 解約が多い | 5,000円の低価格で継続率維持 |
| クレーム（リフォーム品質） | 御社は紹介のみ、責任は会社間 |
| 大家会員が少ない | SEOで会員数を先に増やす |

## 13. まとめ

### ビジネスモデルの構造

```
大家さん（会員）
    ↓ シミュレーター利用で集客
    ↓ 無料会員登録
    ↓
リフォーム会社を探す
    ↓
掲載リフォーム会社一覧を見る
    ↓
問い合わせ・発注

リフォーム会社
    ↓ 月額5,000円で掲載
    ↓ 大家さんからの問い合わせ獲得
    ↓ 成約（御社への報酬なし）
```

### Win-Win-Winの関係

| ステークホルダー | メリット |
|-----------------|----------|
| 大家さん | 信頼できるリフォーム会社が見つかる |
| リフォーム会社 | 月5,000円で大家さんにリーチ |
| 御社 | 月額固定の安定収入 |
