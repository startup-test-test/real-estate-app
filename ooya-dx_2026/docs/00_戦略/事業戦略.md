# 事業戦略

## 1. 結論：なぜ最初から有料化しないのか

### 戦略比較表

| 項目 | 最初から有料化 | 最初は無料（採用） |
|------|---------------|-------------------|
| **集客力** | 弱い | **強い** |
| **SEO効果** | 中 | **高** |
| **口コミ拡散** | 起きにくい | **起きやすい** |
| **初期収益** | あり（少額） | なし |
| **1年後の会員数** | 500人程度 | **1万人目標** |
| **競合との差別化** | 機能勝負 | **価格で圧勝** |
| **将来のB2B価値** | 低い | **高い** |

### 判断理由

| 観点 | 説明 |
|------|------|
| AI開発で低コスト | 有料化しなくても赤字にならない |
| 競合は有料化必須 | 開発投資の回収が必要、御社は身軽 |
| 会員数が資産 | 1万人の大家会員 = B2B広告の価値が高い |
| 「完全無料」は強力な武器 | 口コミ・SEOで広がりやすい |

**→ 最初は完全無料で集客し、会員数を資産にしてからB2Bで収益化**

---

## 2. 基本方針

| 要素 | 内容 |
|------|------|
| AI活用 | 短期間・低コストで26ツール開発済み |
| 完全無料 | 競合との明確な差別化ポイント |
| スピード | 先にシェアを取る |
| 収益化 | 会員数を資産にB2Bで回収 |

### 戦略の流れ

| フェーズ | 期間 | 目標 | 収益 |
|----------|------|------|------|
| Phase 1 | 今〜1年 | 完全無料で集客、会員1万人 | 0円 |
| Phase 2 | 1年後〜 | B2B広告開始 | 50万円/月〜 |
| Phase 3 | 2年後〜 | 必要ならプレミアム機能 | 追加収益 |

---

## 3. なぜこの戦略が成立するのか

### 従来 vs 御社の比較

| 項目 | 従来の競合 | 御社（AI活用） |
|------|------------|----------------|
| 開発期間 | 1ツール数ヶ月 | **数日〜1週間** |
| 開発費/ツール | 100〜500万円 | **ほぼ人件費のみ** |
| 26ツール総コスト | 1億円以上 | **大幅削減** |
| 価格設定 | 有料化で回収必須 | **無料でも成立** |

### 競合が真似できない理由

| 競合タイプ | 無料化できない理由 |
|------------|-------------------|
| 楽待・健美家 | 既存の収益モデルを崩せない |
| 大手不動産会社 | 開発に多額投資済み、回収必要 |
| SaaS系ツール | サブスク収益が前提のビジネスモデル |

---

## 4. 収益化モデル（B2B）

### 広告価値と会員数の関係

| 会員数 | 広告価値 | 販売可能性 |
|--------|----------|------------|
| 〜1,000名 | 低い | 難しい |
| 1,000〜5,000名 | 中程度 | 一部可能 |
| **1万名以上** | **高い** | **十分可能** |
| 5万名以上 | 非常に高い | プレミアム価格 |

### 販売可能な広告枠

| 広告枠 | 想定広告主 | 月額目安 |
|--------|------------|----------|
| バナー広告 | 不動産会社、金融機関 | 5〜30万円 |
| メルマガ広告 | 投資セミナー、書籍 | 10〜50万円/回 |
| 記事広告 | 管理会社、リフォーム会社 | 20〜100万円 |
| リフォーム会社掲載 | リフォーム会社 | 月額3〜10万円/社 |

### 宅建業法との関係

| サービス内容 | 宅建免許 |
|-------------|----------|
| シミュレーター提供 | **不要** |
| 広告スペース販売 | **不要** |
| リフォーム会社マッチング | **不要** |
| 不動産の売買・仲介 | 必要（やらない） |

**→ 広告枠販売モデルは宅建業法に触れない、安心して展開可能**

## 5. 競合分析

### 主要競合のビジネスモデル

| サービス | ユーザー料金 | 収益源 | 特徴 |
|----------|-------------|--------|------|
| 楽待 | 無料 | 不動産会社から掲載料 | 物件ポータル |
| 健美家 | 無料 | 不動産会社から掲載料 | 物件ポータル |
| 不動産投資連合隊 | 無料 | 不動産会社から掲載料 | 物件ポータル |
| smlt.jp | 無料 | 広告収入 | シミュレーター28種 |
| CASIO計算機 | 無料 | 本業のブランディング | 計算機3種 |

### 御社のポジション

| 比較軸 | 競合 | 御社 |
|--------|------|------|
| 主力コンテンツ | 物件情報 | **シミュレーター特化** |
| ツール数 | 少ない or なし | **26種類** |
| ユーザー課金 | なし or あり | **完全無料** |
| 差別化 | 物件数・情報量 | **ツールの充実度** |

## 7. まとめ

### 従来モデル vs 御社モデル

| ステップ | 従来の競合 | 御社 |
|----------|------------|------|
| 1 | 高い開発費 | AI で低コスト開発 |
| 2 | 有料化で回収 | 無料提供可能 |
| 3 | ユーザー離脱 | 大量集客 |
| 4 | 集客苦戦 | B2Bで収益化 |

### 「完全無料」が武器になる4つの理由

| # | 理由 | 説明 |
|---|------|------|
| 1 | 競合との差別化 | 物件サイトではなく「ツール特化」 |
| 2 | SEOで上位表示 | 無料ツールは被リンク・シェアされやすい |
| 3 | 口コミ拡散 | 「無料で使える便利なサイトがある」 |
| 4 | 会員数=資産 | 1万人の大家会員でB2B広告価値が高い |
