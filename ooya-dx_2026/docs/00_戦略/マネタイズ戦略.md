## 2. 広告枠販売（詳細）

### 想定広告主

| 広告主 | ニーズ | 相性 |
|--------|--------|------|
| 不動産売買会社 | 投資家への物件紹介 | ◎ |
| 不動産投資会社 | 新規投資家の獲得 | ◎ |
| 金融機関（銀行・信金） | 不動産ローンの獲得 | ◎ |
| 保険会社 | 火災保険・地震保険 | ○ |
| 不動産クラウドファンディング | 投資家の獲得 | ◎ |

### 広告枠と単価目安

| 広告枠 | 掲載位置 | 月額目安 |
|--------|----------|----------|
| トップバナー | トップページ上部 | 20〜50万円 |
| サイドバナー | 全ページサイドバー | 10〜30万円 |
| ツールページバナー | シミュレーター結果下 | 5〜20万円 |
| メルマガヘッダー | メルマガ冒頭 | 10〜30万円/回 |
| メルマガ本文 | メルマガ内記事 | 20〜50万円/回 |
| 記事広告 | 専用ページ作成 | 30〜100万円/本 |

### 広告価値と会員数の関係

| 会員数 | 広告価値 | 販売可能性 |
|--------|----------|------------|
| 〜1,000名 | 低い | 難しい |
| 1,000〜5,000名 | 中程度 | 一部可能 |
| **1万名以上** | **高い** | **十分可能** |
| 5万名以上 | 非常に高い | プレミアム価格 |

## 9. 優先度まとめ

| 優先度 | マネタイズ | 理由 |
|--------|-----------|------|
| ◎ 高 | アフィリエイト | 初期から可能、低コスト |
| ◎ 高 | HP制作受託 | AI活用で高利益率 |
| ◎ 高 | 広告枠販売（不動産売買会社） | 高単価、ニーズあり |
| ◎ 高 | リフォーム会社掲載料 | 大家との相性抜群 |
| ○ 中 | 管理会社掲載料 | 大家の乗り換えニーズ |
| ○ 中 | 税理士・FP掲載料 | 確定申告シーズンに需要 |
| ○ 中 | セミナー集客代行 | 会員メルマガ活用 |
| ○ 中 | リード販売 | 会員数が増えてから |
| △ 低 | プレミアム課金 | 最後の選択肢 |

## 10. 法律との関係

### 宅建業法

| マネタイズ | 宅建免許 |
|-----------|----------|
| 広告枠販売 | **不要** |
| アフィリエイト | **不要** |
| B2B掲載料 | **不要** |
| HP制作受託 | **不要** |
| リード販売 | **不要** |
| セミナー集客 | **不要** |
| 不動産の売買・仲介 | 必要（やらない） |

### 士業紹介の制限（重要）

| 士業 | 紹介料（成約報酬） | 広告掲載料（固定） |
|------|-------------------|-------------------|
| 税理士 | **NG**（税理士法） | OK |
| 弁護士 | **NG**（弁護士法72条） | OK |
| 司法書士 | **NG**（司法書士法） | OK |
| FP（士業でない） | OK | OK |

**ポイント**：士業からは「紹介料」ではなく「広告掲載料」として受け取る
