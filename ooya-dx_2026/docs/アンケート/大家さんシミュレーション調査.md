# 大家さんシミュレーション実態調査

## 概要

大家さんの投資判断におけるシミュレーション実態を調査し、プレスリリースとして発信することで被リンクを獲得する。

## 調査概要

| 項目 | 内容 |
|------|------|
| 調査対象 | 不動産投資経験のある大家さん |
| 目標サンプル数 | 300〜500件 |
| 設問数 | 10問 |
| 調査方法 | QiQUMO（インターネット調査） |
| 想定費用 | 33,000円〜55,000円（税込） |

## プレスリリース切り口案

| 調査結果（例） | プレスリリース見出し案 |
|--------------|---------------------|
| 「デッドクロスを計算している」が20%以下 | **「大家さんの8割がデッドクロスを未計算」融資破綻リスクの認識不足が浮き彫りに** |
| 表面利回りしか見ていない人が多い | **「表面利回りだけで判断」6割超、実質利回り軽視の実態** |
| シミュレーションツールに不満 | **「計算が面倒」大家さんの7割がシミュレーションに課題** |
| Excelで自作している人が多い | **大家さんの投資判断、いまだ「Excel頼み」が主流** |
| 売却時の手取りを計算していない | **「出口戦略なき投資」売却シミュレーション未実施が過半数** |

## 費用シミュレーション（QiQUMO）

計算式：サンプル数 × 設問数 × 11円（税込）

| サンプル数 | 10問の場合 |
|-----------|-----------|
| 300件 | 33,000円 |
| 500件 | 55,000円 |
