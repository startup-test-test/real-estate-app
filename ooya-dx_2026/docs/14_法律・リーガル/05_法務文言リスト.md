# 法務文言リスト（禁止・注意ワード集）

コンテンツ作成時に避けるべき文言・表現をまとめたリストです。

---

## 1. サービス表現の言い換え（重要）

「投資」という表現を避け、「賃貸経営」に統一する。

| 禁止ワード | 言い換え |
|-----------|---------|
| 不動産投資 | 賃貸経営、不動産賃貸経営 |
| 投資判断 | 経営判断、意思決定 |
| 投資家 | オーナー、大家、賃貸経営者 |
| 投資物件 | 収益物件、賃貸物件 |
| 投資用不動産 | 収益不動産、賃貸用不動産 |
| 投資収益 | 賃貸収益、運用収益 |
| 投資利回り | 賃貸利回り、収益利回り |

**理由：** 「投資」という表現は金融商品取引法上の投資助言業に該当するリスクを高めるため。

---

## 2. 数値目安に対する評価語（禁止）

指標の数値に対して以下の評価語を使用しない。

| 禁止ワード | 理由 |
|-----------|------|
| 低い | 判断を誘導する評価表現 |
| やや低い | 同上 |
| 標準的 | 同上 |
| 良好 | 同上 |
| 優良 | 同上 |
| 非常に高い | 同上 |
| 高い | 同上（リスク評価を含む場合） |
| おすすめ | 助言に該当する可能性 |
| 買い時 | 同上 |
| 売り時 | 同上 |

**NGパターン例：**
```
| IRR | 評価 |
| 3%未満 | 低い |
| 8〜12% | 良好 |
| 12%以上 | 優良 |
```

**OKパターン：**
- 計算式の説明のみ記載
- 「○○%程度とされています」（出典明記必須）

## 5. 税理士業務に関する禁止ワード

税理士の独占業務（税理士法第2条・第52条）を想起させる表現は避ける。

| 禁止ワード | 言い換え例 |
|-----------|-----------|
| 税務相談 | 税制の一般的な解説 |
| 節税アドバイス | 軽減制度の紹介 |
| 確定申告の代行 | （提供しない） |
| あなたの税額は○○円です | 税額の計算方法を解説 |
| ○○すれば節税できます | ○○という制度があります |
| 税務申告書の作成 | （提供しない） |
| 税務代理 | （提供しない） |

**NG行為：**
- 個別具体的な税額の算定
- 特定ユーザーへの税務アドバイス
- 確定申告書の作成代行

**OK行為：**
- 税制の一般的な仕組みの解説
- 計算式・税率表の掲載
- シミュレーターによる概算計算（免責事項付き）

## 12. 誇大表現・保証表現

景表法違反リスクのある表現。

| 禁止ワード | 理由 |
|-----------|------|
| 業界No.1 | 根拠なき誇大表現 |
| 最高精度 | 同上 |
| 必ず儲かる | 保証表現 |
| 絶対に損しない | 同上 |
| 確実に○○ | 同上 |
| 利回り保証 | 同上（→「想定利回り」に） |
| 節税確約 | 同上（→「軽減効果の目安」に） |
| 間違いない | 保証表現 |

---
