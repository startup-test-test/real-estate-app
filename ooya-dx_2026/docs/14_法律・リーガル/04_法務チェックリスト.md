# 法務チェックリスト

文章・シミュレーター・ページを作成した後に確認するチェックリストです。
弁護士（宮岡遼氏）のフィードバックに基づいて作成。

---

## 1. コンテンツ（文章・解説）作成時

### 必須チェック

- [ ] **「教育目的」という表現を使用していない**
  - 理由: 助言に該当する可能性を高めるため削除推奨
  - OK: 「参考情報」「参考目的」

- [ ] **特定物件への投資判断を含んでいない**
  - NG: 「○○マンションは利回り8%なので今買うべき」
  - NG: 「この物件は投資に適している」「購入すべき」「売却すべき」
  - OK: 一般的な情報分析や自己判断のためのツール提供

- [ ] **個別具体的な税務相談をしていない**
  - NG: 「あなたの場合は○○円の控除が受けられます」
  - NG: 特定ユーザーの確定申告内容を指導
  - OK: 一般的な税制の解説

- [ ] **根拠のない誇大表現を使用していない**
  - NG: 「業界No.1の精度」（根拠なし）
  - NG: 「必ず儲かる」「絶対に損しない」
  - OK: 具体的な根拠がある表現

- [ ] **専門家への相談を誘導している**
  - 必須: 「最終的な判断は専門家にご相談ください」等の文言

### OK/NG表現クイックリファレンス

| NG表現 | OK表現 |
|--------|--------|
| 教育目的 | 参考情報、参考目的 |
| この物件は買うべき | 一般的な収益計算の方法を解説 |
| あなたの税額は○○円 | 税額の計算方法を解説 |
| 業界No.1 | （根拠があれば可） |

  ```
  最終更新日: YYYY年MM月DD日
  ```

- [ ] **「●年●月時点での法律に基づいています」と記載している**
  - より安全性を高める記載

- [ ] **計算式を表示している（推奨）**
  - シミュレーション結果と一緒に計算式も出力するとより安全

- [ ] **取引媒介機能を設けていない**
  - NG: 購入申し込みボタン
  - NG: 取引相手との連絡機能
  - NG: 特定の売主・買主を紹介する機能

- [ ] **最終更新日を記載している**

### シミュレーター免責事項テンプレート

```markdown
## 免責事項

- 本シミュレーターの計算結果は概算値であり、実際の金額は異なる場合があります。
- 本計算は YYYY年MM月時点の法律・税制に基づいています。
- 本サイトの情報により生じた損害について、当サイト運営者は一切の責任を負いません。

最終更新日: YYYY年MM月DD日
```

## 4. 広告・アフィリエイト関連

### 必須チェック

- [ ] **積極的な購入・売却の推奨をしていない**
  - NG: 「○○で売却すべき」「今すぐ査定を」と強く勧める
  - OK: 広告スペースとしてリンクを設置

- [ ] **仲介行為に該当していない**
  - NG: 成約したら手数料をもらう（仲介手数料）
  - OK: 広告掲載料、アフィリエイト報酬（訪問・登録ベース）

- [ ] **個別の契約に介入していない**
  - 原則: 個別の契約にはタッチせず距離をとっておく

### アフィリエイトOK/NG

| OK | NG |
|----|-----|
| サイト訪問数に応じた報酬 | 成約に応じた仲介手数料 |
| 無料査定申込完了の成果報酬 | 特定物件の売却を積極的に勧める |
| 広告枠として物件掲載 | 売主・買主の間に入って契約成立 |

## 6. 公開前最終チェック

### 宅建業法チェック

- [ ] 「この物件を購入すべき」「売却すべき」と断定していない
- [ ] 「成約したら手数料をください」等の仲介報酬を求めていない
- [ ] 購入申し込みボタンを設置していない
- [ ] 取引相手との連絡機能を設けていない
- [ ] 特定の売主・買主・仲介業者を紹介する機能がない
- [ ] 売買契約の成立に関与していない

### 金融商品取引法チェック

- [ ] 「この物件は買いです」と断定していない
- [ ] 「今が売り時」「今すぐ買うべき」と断定していない
- [ ] 「利回り○%なので投資すべき」と断定していない
- [ ] 「必ず儲かる」「絶対に損しない」と断定していない
- [ ] 投資判断はユーザー自身が行う旨を明記している

### 税理士法チェック

- [ ] 「あなたの税額は○○円です」と個別回答していない
- [ ] 「あなたの場合は○○円の控除が受けられます」と個別回答していない
- [ ] 特定ユーザーの確定申告内容を指導していない
- [ ] 「詳細は税理士にご相談ください」と誘導している
- [ ] 一般的な税制解説に留めている

### 景品表示法チェック

- [ ] 「業界No.1」「日本一」等を根拠なく使用していない
- [ ] 「最高精度」「最も正確」等を根拠なく使用していない
- [ ] 「必ず」「絶対」「100%」等の断定表現を使用していない
- [ ] 誇大表現には具体的な根拠・出典を明記している
- [ ] 実際より著しく良く見せる表示をしていない

### 消費者契約法チェック

- [ ] 不動産事業用利用の建て付けになっている
- [ ] 利用規約で事業用利用の承諾を取っている

### 法改正情報の掲載チェック

**⚠️ 法改正の解釈は弁護士の領域です。以下のルールを厳守してください。**

#### 基本原則

> **「法律がこう変わりました」という事実の紹介 = OK**
> **「だからあなたはこうすべき」というアドバイス = NG**

事実を報告するだけなら資格は不要です。判断・適用のアドバイスは弁護士の領域です。

#### 掲載OK（情報提供・紹介）

- [ ] 公式ソース（国土交通省・国税庁等）のエビデンスリンクがある
- [ ] 本人（運営者）が内容を確認・理解できている
- [ ] 参照リンクを明記している
- [ ] 事実の紹介に留めている（「○○が改正されました」）
- [ ] 詳細は専門家への確認を誘導している

#### 掲載NG（法的アドバイス）

- [ ] エビデンスが不明確・不透明な内容を掲載していない
- [ ] 本人が確認できない内容を掲載していない
- [ ] 法改正の「解釈」や「適用判断」をしていない
- [ ] 「この法改正を使うべき」等のアドバイスをしていない
- [ ] 「あなたの場合は○○」等の個別判断をしていない

| OK表現（情報提供） | NG表現（法的アドバイス） |
|-------------------|------------------------|
| 「2024年7月に法改正されました」 | 「この法改正を使うべきです」 |
| 「上限が33万円になりました」 | 「あなたの場合は○○円払えばいい」 |
| 「詳細は不動産会社にご確認ください」 | 「この特例を適用すれば得します」 |

**判断に迷う場合は、法改正情報の掲載を見送り、弁護士に確認してください。**

### 必須要素チェック

| 要素 | 具体的な内容 | 確認 |
|------|-------------|------|
| 免責事項 | 「本シミュレーターの計算結果は概算値であり...」 | [ ] |
| 法律基準日 | 「●年●月時点での法律に基づいています」 | [ ] |
| 最終更新日 | 「最終更新日: YYYY年MM月DD日」 | [ ] |
| 専門家誘導 | 「最終的な判断は専門家にご相談ください」 | [ ] |
| 利用規約リンク | フッター等に設置 | [ ] |
| 計算式表示 | シミュレーターの場合、計算ロジックを表示（推奨） | [ ] |

## 7. 判断に迷った場合

以下の場合は弁護士に確認:

1. **新しい機能の追加**
   - 取引の成立に向けた連絡・交渉を代行または支援する機能
   - ユーザーに特定の売主・買主・仲介業者を紹介する機能

2. **表現に自信がない場合**
   - 景表法上の問題がありそうな表示
   - 投資助言に該当しそうな表現

3. **新しいビジネスモデル**
   - 収益構造が変わる場合
   - 報酬の受け取り方が変わる場合
