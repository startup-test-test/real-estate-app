# 不動産関連サービスで法律違反になりうるケース

弁護士に確認する際の参考として、一般的に注意すべきポイントをまとめます。

---

## 1. 宅建業法違反（最も注意が必要）

| 違反行為 | 説明 |
|---------|------|
| 仲介行為 | 売主・買主の間に入って契約を成立させ、報酬を得る |
| 反復継続 | 自ら不動産売買を反復継続して行う |

**NG例:**
- 「この物件を紹介するので、成約したら手数料をください」

**チェックリスト:**
- [ ] 仲介行為（売主・買主の間に入って契約成立）をしていない
- [ ] 仲介手数料を受け取っていない
- [ ] 特定物件への「購入すべき」「売却すべき」等の助言をしていない
- [ ] 購入申し込みボタン、取引相手との連絡機能を設けていない

---

## 2. 金融商品取引法違反

| 違反行為 | 説明 |
|---------|------|
| 投資助言業 | 「この物件は買いです」と個別具体的な投資判断を提供して報酬を得る |

**NG例:**
- 「○○マンションは利回り8%なので今買うべき」と有料で助言

**チェックリスト:**
- [ ] 特定物件への個別具体的な投資判断を提供していない
- [ ] 「この物件は買い」「今が売り時」等の断定的な助言をしていない
- [ ] 投資判断の最終決定はユーザー自身であることを明記している

---

## 3. 税理士法・司法書士法違反

| 違反行為 | 説明 |
|---------|------|
| 個別具体的な税務相談 | 「あなたの場合は○○円の控除が受けられます」 |

**NG例:**
- 特定ユーザーの確定申告内容を指導する

**チェックリスト:**
- [ ] 個別具体的な税務相談をしていない
- [ ] 「あなたの税額は○○円」等の個別回答をしていない
- [ ] 一般的な税制解説に留めている
- [ ] 「詳細は税理士にご相談ください」と誘導している

---

## 4. 景品表示法違反

| 違反行為 | 説明 |
|---------|------|
| 優良誤認・有利誤認 | 実際より著しく良く見せる表示 |

**NG例:**
- 「業界No.1の精度」（根拠なし）

**チェックリスト:**
- [ ] 根拠のない「No.1」「業界最高」等の表現を使用していない
- [ ] 「必ず儲かる」「絶対に損しない」等の断定表現を使用していない
- [ ] 誇大表現には具体的な根拠・出典を明記している

## 大家DXで安全なライン

| OK（問題なし） | NG（違反リスク） |
|---------------|-----------------|
| 一般的な計算ツール提供 | 個別の投資判断を有料で助言 |
| 広告枠として物件掲載 | 仲介として報酬を得る |
| 一般的な税制解説 | 個別の税務相談 |
| 「専門家に相談を」と誘導 | 最終判断を代行 |

---

## まとめ

弁護士に「どのようなケースが違反になるか」を確認しておくと、今後のサービス拡大時に判断基準になります。

**重要な原則:**
- 個別の契約にはタッチせず距離をとっておく
- 一般的な情報提供に留め、個別具体的な助言は専門家へ誘導
