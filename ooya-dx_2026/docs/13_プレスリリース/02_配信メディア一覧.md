# プレスリリース配信メディア一覧

不動産・賃貸経営関連のプレスリリースを収集できるメディア一覧です。

## 主要プレスリリース配信サービス

### 1. PR TIMES（ピーアールタイムズ）

- **URL**: https://prtimes.jp/
- **メディア登録**: https://prtimes.jp/main/registmedia/form
- **特徴**: 国内最大級のプレスリリース配信サービス
- **月間PV**: 約8,000万PV
- **掲載企業数**: 約85,000社以上
- **不動産カテゴリ**: あり
- **優先度**: ★★★★★

### 2. @Press（アットプレス）

- **URL**: https://www.atpress.ne.jp/
- **メディア登録**: https://www.atpress.ne.jp/media_register
- **特徴**: 配信代行型、提携メディアへの配信力が強い
- **掲載保証**: あり（有料プランによる）
- **不動産カテゴリ**: あり
- **優先度**: ★★★★☆

### 3. ValuePress!（バリュープレス）

- **URL**: https://www.value-press.com/
- **メディア登録**: https://www.value-press.com/media/register
- **特徴**: 中小企業向け、リーズナブル
- **月間配信数**: 約3,000件
- **不動産カテゴリ**: あり
- **優先度**: ★★★☆☆

### 4. 共同通信PRワイヤー

- **URL**: https://kyodonewsprwire.jp/
- **メディア登録**: 要問い合わせ
- **特徴**: 共同通信グループ、信頼性が高い
- **配信先**: 全国紙、地方紙、テレビ局
- **優先度**: ★★★★☆

### 5. Dream News（ドリームニュース）

- **URL**: https://www.dreamnews.jp/
- **メディア登録**: https://www.dreamnews.jp/media/
- **特徴**: 低価格、スタートアップ向け
- **月額料金**: 無料プランあり
- **優先度**: ★★★☆☆

### 6. NEWSCAST（ニュースキャスト）

- **URL**: https://newscast.jp/
- **メディア登録**: https://newscast.jp/media
- **特徴**: 無料で利用可能、SNS連携
- **配信先**: Yahoo!ニュース連携あり
- **優先度**: ★★★☆☆

### 7. Bridge（ブリッジ）

- **URL**: https://thebridge.jp/
- **特徴**: スタートアップ・テック特化
- **対象**: IT・テクノロジー企業向け
- **優先度**: ★★☆☆☆（不動産特化ではない）

## 登録優先順位

### フェーズ1（まず登録）
1. **PR TIMES** - 最大手、必須 ✅ 申請済み
2. **@Press** - 配信力が高い
3. **ValuePress!** - 幅広いカバー

### フェーズ2（業界特化）
4. **健美家** - 不動産投資家へのリーチ
5. **楽待** - 不動産投資家へのリーチ
6. **不動産流通研究所** - 業界関係者へのリーチ

### フェーズ3（補完）
7. **Dream News** - 無料で追加カバー
8. **NEWSCAST** - SNS連携
9. **共同通信PRワイヤー** - 信頼性重視の場合

## 転載時の注意事項

1. **著作権の確認**: プレスリリースの転載許可を確認
2. **出典明記**: 配信元・企業名を明記
3. **改変禁止**: 原則として内容の改変は不可
4. **リンク設置**: 元記事へのリンクを設置
5. **画像利用**: 提供画像の利用可否を確認
