### 内部リンク構造（トピッククラスター）

```
        /tools/（ピラー）
            │
            ↓ 一覧リンク
    /tools/brokerage/（ツール）
            │
            ↕ 相互リンク
    /media/.../brokerage-guide/（記事）
            │
            ↑ 一覧リンク
        /media/（ピラー）
```

**ポイント：**
- ツールとメディアは分離して管理
- 同じトピックのツールと記事は相互リンク
- 各ピラーページから個別ページへリンク

### 内部リンク運用方針

#### フェーズ1: シミュレーターのみ（現在）

記事ページがない段階では、シミュレーターページに解説を含める。

```
/tools/brokerage/
├─ シミュレーター本体
├─ 早見表
├─ 解説セクション（仲介手数料とは、計算方法、etc.）
├─ FAQ
└─ 関連ツール
```

#### フェーズ2: 記事ページ追加時

記事ページを追加したら、相互リンクを設定する。

**シミュレーター → 記事**
```html
<section>
  <h2>仲介手数料について詳しく知る</h2>
  <a href="/media/.../brokerage-guide/">仲介手数料の基礎知識</a>
  <a href="/media/.../brokerage-negotiation/">仲介手数料の相場と交渉術</a>
</section>
```

**記事 → シミュレーター**
```html
<div class="tool-cta">
  <p>今すぐ仲介手数料を計算する</p>
  <a href="/tools/brokerage/">シミュレーターを使う →</a>
</div>
```

**注意点：**
- シミュレーターの解説は削除しなくてOK（重複しても問題なし）
- 記事は別アングルで深掘り（相場、交渉術、無料の落とし穴など）
- 相互リンクでカニバリゼーションを防止

### タイトルタグのルール（CTR向上）

#### 基本フォーマット
```
[対象]の[計算内容]を[時間]で無料計算｜[付加価値]
```

#### ルール詳細

| 項目 | ルール | 例 |
|------|--------|-----|
| 文字数 | 27〜30文字以内 | PC/SP両対応 |
| 「無料」の位置 | 「無料計算」「無料で計算」 | 法的リスク回避 |
| 具体性 | 「10秒で」「瞬時に」 | 手軽さをアピール |
| 付加価値 | 「早見表付き」「税込対応」 | 差別化要素 |

#### 避けるべき表記

| NG例 | 理由 |
|------|------|
| 【無料】仲介手数料... | 「手数料無料」と誤認される可能性（景品表示法リスク） |
| ○○シミュレーター｜大家DX | ブランド名より機能・メリットを優先 |

## 早見表の活用

全シミュレーターに**早見表**を追加し、コンテンツ量とSEO効果を向上させる。

### 早見表のメリット

| 効果 | 説明 |
|------|------|
| コンテンツ量増加 | 5,000字以上達成しやすい |
| ロングテール獲得 | 「仲介手数料 3000万円」などで流入 |
| 滞在時間UP | 表を見て比較するユーザーが増える |
| ユーザー体験 | 計算が苦手な人にも優しい |

### 各シミュレーターの早見表設計

| シミュレーター | 早見表の軸 | 表示例 |
|--------------|----------|--------|
| 仲介手数料 | 売買価格 | 1,000万円→39.6万円 |
| 贈与税 | 贈与額 | 500万円→48.5万円 |
| 譲渡所得税 | 譲渡益 | 1,000万円→203万円 |
| 不動産取得税 | 固定資産税評価額 | 2,000万円→30万円 |
| 印紙税 | 契約金額 | 1,000万円→1万円 |
| 登録免許税 | 固定資産税評価額 | 2,000万円→40万円 |
| 相続税 | 遺産総額 | 1億円→1,220万円 |
| 減価償却 | 建物価格×構造 | RC・5,000万円→年間106万円 |

### 実装方針

- 主要価格帯を**100万円〜500万円刻み**で表示
- 税込金額を表示（ユーザーが最も知りたい情報）
- 共通コンポーネントとして実装し、再利用可能に

## 細分化の判断基準

派生ページを作るかどうかは、以下で判断：

| 条件 | 判断 |
|------|------|
| 派生キーワードの検索数 > 1,000/月 | 別ページを検討 |
| 派生キーワードの検索数 < 1,000/月 | メインページ内で対応 |
| 派生ページの流入がメインの5%未満 | 不要 |

**例：仲介手数料**
- メイン（速算式）: 35,381 → ✅ 必須
- レーマン方式: 154 → ❌ 不要（メイン内で対応）
- 円単位計算: 80 → ❌ 不要（トグルで対応）
