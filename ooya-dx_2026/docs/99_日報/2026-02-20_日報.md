### 9. free.mdx（無料の仕組みページ）の完全削除

**⑤ 無料の仕組み**ページを削除し、10ページ構成 → **9ページ構成**に変更。

#### 削除理由

1. 筆者の実体験に基づかない — 仲介手数料「無料」の会社は見たことがない（半額は存在する）
2. 記事内容に根拠の薄い主張が多い — 囲い込みリスク、追加費用請求など裏付けなし
3. 検索ユーザーの質が低い — 「無料」を求めるユーザーはCV率が低い
4. 著者のポリシーと矛盾 — 値切らない方がいいという立場と方向性が合わない

#### リンク整理

| ファイル | 対応 |
|---------|------|
| `guide.mdx` | free.mdx へのCTALink＋FAQ参照を削除 |
| `negotiation.mdx` | free.mdx へのCTALink＋FAQ参照を削除 |
| `rental.mdx` | free.mdx へのCTALink＋FAQ参照を削除 |
| `market-rate.mdx` | free.mdx へのCTALink＋FAQ参照を削除 |
| `housing-loan.mdx` | free.mdx へのCTALink＋FAQ参照を削除 |
| `real-estate.mdx` | free.mdx への参照を削除 |
| `price-examples.mdx` | free.mdx へのCTALinkを削除 |

計 **12箇所のリンク** を7ファイルから削除。
