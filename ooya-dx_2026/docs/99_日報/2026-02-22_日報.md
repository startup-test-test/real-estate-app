### 13. コンテンツ戦略の転換：「大手型解説」→「実体験ベース」

Geminiとの戦略会議を通じて、コンテンツの方向性を大幅に転換。

#### 課題認識

- 現在のタイトル（「徹底解説」「完全ガイド」）は大手メディア（SUUMO、HOME'S）と同じ土俵
- 個人がドメインパワーで大手に勝つのは困難
- 法律解説の網羅性では差別化できない

#### 方針転換

| 項目 | 変更前 | 変更後 |
|------|--------|--------|
| コンテンツの主役 | 法律解説・計算方法 | **著者の実体験データ** |
| 差別化ポイント | 網羅性 | **7軒の取引実績（一次情報）** |
| タイトル | 「〜を徹底解説【2026年版】」 | 「【実体験KW】｜実体験ベースの表現」 |
| 立ち位置 | 専門メディア風 | **個人大家の実録ブログ** |

#### タイトル変更案（SEO KW維持＋実体験）

| ファイル | 現タイトル | 変更案 |
|---------|-----------|--------|
| guide.mdx | 仲介手数料とは？…完全ガイド | 【実体験】仲介手数料とは？｜7軒購入・1件売却した大家が「損しない仕組み」を解説 |

## 発見した気づき・方針

| 気づき | 判断 |
|--------|------|
| 実体験データを整理すると「不動産投資の始め方」（初心者向け）がブログの方向性として最適 | 今後のコンテンツ設計の軸にする |
| 売買3%、賃貸1ヶ月は「相場」というより「当たり前」 | 読者が本当に知りたいのは「安くする方法」 |
| 農地売買など経験のないテーマは削除すべき | E-E-A-T重視で実体験ベースに集中 |
| 計算式を入れると「なぜその金額になるか」が直感的にわかる | 全ページの早見表に統一的に適用 |
| **大手と同じ「徹底解説」型では勝てない** | 実体験データ（一次情報）で差別化する |
| 法律の断定は資格なしではリスクが高い | 「一般的です」「とされています」と表現を柔らかくする |
| 売買価格の端数（6,980万円等）は物件特定リスクがある | 100万円単位に丸め、仲介手数料は実額で公開 |
| Geminiの「構造化データ＋固有名詞＋画像alt」戦略 | Person JSON-LDは実装済み。請求書写真の検討へ |
| ExperienceTableコンポーネント化で保守性向上 | データ1箇所修正で全ページ反映 |
