## 今後の開発アイデア

### 新規ツール: キャッシュフロー計算機

| 項目 | 内容 |
|---|---|
| 名称 | キャッシュフロー計算機 |
| 位置付け | 簡易版CF計算ツール（物件スクリーニング用） |
| 差別化 | 賃貸経営シミュレーター = 高機能・詳細分析 |
|  | キャッシュフロー計算機 = シンプル・即時計算 |
| SEO効果 | 「キャッシュフロー 計算」検索キーワードに対応 |

### 名称の選定理由
1. SEOで「キャッシュフロー 計算」で検索する人に刺さる
2. 「シミュレーター」との違いが明確（計算機 = シンプル、シミュレーター = 詳細）
3. 何ができるか一目でわかる
