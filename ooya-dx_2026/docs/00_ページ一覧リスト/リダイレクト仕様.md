# リダイレクト仕様

> 最終更新: 2026-02-05

## 概要

Next.js の `next.config.mjs` でリダイレクトを設定しています。
すべて **301リダイレクト（恒久的）** です。

## 用語集リダイレクト（2026-02-05 追加）

用語集ページを廃止し、関連する計算ツールへリダイレクト。

### 廃止理由

- 内容が薄く、大手サイトに勝てない
- 「〇〇とは」系キーワードは大手（SUUMO、HOME'S等）が上位独占
- 計算ツールページに集中する方が効果的

## 設定ファイル

```
ooya-dx_2026/next.config.mjs
```

### 設定例

```javascript
async redirects() {
  return [
    {
      source: '/glossary/ccr',
      destination: '/tools/ccr',
      permanent: true,  // 301リダイレクト
    },
    // ...
  ];
},
```

---

## 確認方法

### ローカル環境

```bash
npm run dev
# ブラウザで /glossary/ccr にアクセス → /tools/ccr にリダイレクトされることを確認
```

### 本番環境

```bash
curl -I https://ooya.tech/glossary/ccr
# HTTP/2 301 と Location: /tools/ccr が返ることを確認
```
