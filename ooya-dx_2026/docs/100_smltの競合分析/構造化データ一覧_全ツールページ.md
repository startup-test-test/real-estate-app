## 共通コンポーネント

**ファイル:** `components/tools/ToolStructuredData.tsx`

全ツールページで共通のコンポーネントを使用。以下の構造化データを自動生成：

| スキーマ | 説明 | 条件 |
|----------|------|------|
| **SoftwareApplication** | Googleに「計算ツール」と認識させる | 常に生成 |
| **BreadcrumbList** | パンくずリスト（ホーム → ツール一覧 → 各ツール） | 常に生成 |
| **WebPage** | 公開日・更新日情報 | datePublished指定時のみ |
| **FAQPage** | FAQ構造化データ（リッチリザルト対応） | faqItems指定時のみ |

### コンポーネントのProps一覧

| prop | 型 | 必須 | 説明 |
|------|-----|------|------|
| `name` | string | ○ | ツール名（例: `仲介手数料計算シミュレーター`） |
| `description` | string | ○ | ツールの説明文 |
| `toolPath` | string | ○ | ツールのパス（例: `/tools/brokerage`） |
| `datePublished` | string | - | 公開日（例: `2026-01-15`）→ WebPage生成 |
| `dateModified` | string | - | 更新日（例: `2026-02-05`） |
| `faqItems` | FaqItem[] | - | FAQリスト → FAQPage生成 |
| `featureList` | string[] | - | ツールの主要機能リスト（Gemini提案で追加） |
| `image` | string | - | ツールのスクリーンショット画像パス（Gemini提案で追加） |

### SoftwareApplication 共通設定

```json
{
  "@type": "SoftwareApplication",
  "applicationCategory": "FinanceApplication",
  "operatingSystem": "All",
  "offers": { "price": "0", "priceCurrency": "JPY" },
  "provider": {
    "@type": "Organization",
    "name": "大家DX",
    "url": "https://ooya.tech",
    "logo": "https://ooya.tech/img/logo_250709_2.png"
  },
  "featureList": "（featureList prop指定時のみ出力、カンマ区切り文字列）",
  "image": "（image prop指定時のみ出力、フルURL）"
}
```

### Geminiフィードバック反映内容（2026-02-08追加）

| 改善項目 | 対応内容 |
|----------|----------|
| **provider.logo** | Organization に `logo` プロパティ追加（E-E-A-T強化） |
