# 級数・フォントサイズ仕様

## 概要

PC/SP（スマートフォン）で異なるフォントサイズを、Tailwindのレスポンシブクラスで一元管理する。
共通コンポーネント内に定義することで、1箇所の修正で全ページに反映される。

---

## ブレークポイント

| プレフィックス | 幅 | 用途 |
|----------------|-----|------|
| （なし） | 0〜639px | スマートフォン（SP） |
| `sm:` | 640px〜 | タブレット |
| `md:` | 768px〜 | 小型PC |
| `lg:` | 1024px〜 | PC |
| `xl:` | 1280px〜 | 大型PC |

---

## ページレイアウト系

### H1 ページタイトル

**コンポーネント:** `ToolPageHeader.tsx`

```tsx
<h1 className="text-xl sm:text-2xl md:text-3xl font-bold text-gray-900 leading-tight">
```

| デバイス | Tailwindクラス | サイズ |
|----------|----------------|--------|
| SP | `text-xl` | 20px (1.25rem) |
| タブレット | `sm:text-2xl` | 24px (1.5rem) |
| PC | `md:text-3xl` | 30px (1.875rem) |

---

### H2 セクション見出し

**コンポーネント:** 各ページ内で定義


|----------|----------------|--------|
| SP | `text-base` | 16px (1rem) |

## 変更例

### 全体的に大きくしたい場合

```tsx
// CalculatorInputField.tsx
- className="text-xl sm:text-4xl"
+ className="text-2xl sm:text-5xl"
```

→ 全27ツールの入力値が大きくなる

### SPだけ大きくしたい場合

```tsx
// CalculatorMainResult.tsx
- className="text-2xl sm:text-5xl"
+ className="text-3xl sm:text-5xl"
```

→ SPのメイン結果値だけ24px→30pxに変更
