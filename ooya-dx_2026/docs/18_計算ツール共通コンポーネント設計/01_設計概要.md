# 計算ツール共通コンポーネント設計書

## 概要

全24種類の計算ツール（シミュレーター）ページで共通利用するコンポーネントの設計書。
WordPress的な統一感のあるレイアウト・デザインを実現する。

---

## 現状の課題

1. 各ツールページで個別にレイアウトを実装している
2. 右カラム（サイドバー）のコードが重複している
3. シミュレーターUIのデザインが統一されていない
4. デザイン変更時に全ページを修正する必要がある

---

## 目標

1. **レイアウトの共通化** - 2カラム構成（左：メインコンテンツ、右：サイドバー）
2. **サイドバーの共通化** - ツール一覧・CTAを全ページで統一
3. **シミュレーターUIの共通化** - 入力・結果表示のデザイン統一
4. **レスポンシブ対応** - PC/SPをTailwindで一元管理

---

## レスポンシブ方針

**PC/SPを分けず、Tailwindのレスポンシブクラスで対応**

| ブレークポイント | 幅 | 用途 |
|------------------|-----|------|
| デフォルト | 〜639px | スマホ（SP） |
| `sm:` | 640px〜 | タブレット |
| `md:` | 768px〜 | 小型PC |
| `lg:` | 1024px〜 | PC（2カラム切替） |
| `xl:` | 1280px〜 | 大型PC |

### 主な対応パターン

```tsx
// フォントサイズ
className="text-sm sm:text-base lg:text-lg"

// パディング
className="p-3 sm:p-4 lg:p-6"

// 表示切替
className="hidden lg:block"     // PCのみ
className="lg:hidden"           // SPのみ

// グリッド
className="lg:grid lg:grid-cols-[1fr_300px] lg:gap-10"
```
