## 施策詳細

### 施策1: ツールページに関連記事セクション追加

**現状:**
- RelatedTools（同カテゴリツール）は実装済み
- 関連するメディア記事へのリンクがない

**改善案:**
```
┌─────────────────────────────────────┐
│ [計算ツール] IRR（内部収益率）      │
│                                     │
│ [計算機能]                          │
│                                     │
│ [関連ツール] ← 実装済み             │
│ ・NPV ・DCF法 ・CCR                 │
│                                     │
│ [関連記事] ← 新規追加               │
│ ・IRRとは？不動産投資での活用法     │
│ ・NPVとIRRの違いを徹底解説         │
│ ・収益物件の評価方法5選            │
└─────────────────────────────────────┘
```

**実装方法:**
1. 記事メタデータに `relatedTools` フィールド追加
2. `RelatedArticles` コンポーネント作成
3. ツールページに組み込み

**期待効果:**
- ツール→記事への回遊増加
- 滞在時間向上
- 直帰率低下

### 施策2: 用語ページに関連ツール・記事リンク強化

**現状:**
- `relatedTools` フィールドはあるが活用が限定的
- 用語ページからの発リンクが少ない

**改善案:**
```
┌─────────────────────────────────────┐
│ [用語] IRR（内部収益率）            │
│                                     │
│ [定義・解説]                        │
│                                     │
│ [この用語を使うツール] ← 強化       │
│ ・IRR計算ツール                     │
│ ・NPV計算ツール                     │
│                                     │
│ [関連記事] ← 新規追加               │
│ ・IRRとは？不動産投資での活用法     │
│                                     │
│ [関連用語] ← 新規追加               │
│ ・NPV ・DCF ・キャップレート       │
└─────────────────────────────────────┘
```

**実装方法:**
1. 用語メタデータに `relatedArticles`, `relatedTerms` 追加
2. 用語ページテンプレート更新
3. 既存用語データの更新

**期待効果:**
- 用語集の孤立解消
- SEOリンクジュース分配
- 用語集からの回遊増加

### 施策6: 用語自動リンク（記事内）

**現状:**
- 記事内での用語リンクは手動
- 用語集との連携が弱い

**改善案:**
記事本文内の用語を自動検出し、用語集へリンク

```
記事本文:
「IRRは内部収益率と呼ばれ、NPVがゼロになる割引率です」
        ↓ 自動変換
「[IRR](/glossary/irr)は内部収益率と呼ばれ、[NPV](/glossary/npv)がゼロになる割引率です」
```

**実装方法:**
1. 用語リストを取得
2. MDX処理時に自動リンク挿入
3. 初出のみリンク（重複防止）

**注意点:**
- 処理負荷が高い
- 誤リンクの可能性
- 優先度は低め
