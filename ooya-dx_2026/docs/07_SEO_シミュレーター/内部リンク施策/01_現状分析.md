## 4. 強みと課題

### 強み

| 項目 | 詳細 |
|------|------|
| 一元管理 | `lib/navigation.ts`で全リンク管理 |
| 自動生成 | ツール・記事・用語をファイルシステムから自動検出 |
| クロスリンク | トップページでツール→メディア→用語集を露出 |
| カテゴリ分類 | ツール5分類、用語50音分類 |
| SEO最適化 | 全ページにパンくず + 構造化データ |
| RelatedTools | 同カテゴリツールを自動表示 |

### 課題

| 項目 | 詳細 | 影響 |
|------|------|------|
| ツール→記事リンク不足 | ツールページから関連記事へのリンクがない | 回遊率低下 |
| 記事→ツールリンク不足 | 記事内でのツール紹介が手動・不足 | CV機会損失 |
| 用語集の孤立 | 用語ページからの発リンクが少ない | SEO効果低下 |
| カテゴリページなし | `/tools/category/tax` のようなページがない | ロングテールSEO弱い |
| 記事間リンク不足 | 関連記事への自動リンクがない | 滞在時間短縮 |
