## 法人税等の構成

不動産法人が負担する「法人税等」は、複数の税金で構成される。

| 税金の種類 | 区分 | 課税標準 |
|------------|------|----------|
| 法人税 | 国税 | 課税所得 |
| 地方法人税 | 国税 | 法人税額 |
| 法人住民税（法人税割） | 地方税 | 法人税額 |
| 法人住民税（均等割） | 地方税 | 定額（資本金・従業員数による） |
| 法人事業税 | 地方税 | 課税所得 |
| 特別法人事業税 | 国税 | 法人事業税額 |

## 計算ロジック

### 1. 法人税（国税）

#### 中小法人の税率（資本金1億円以下）

| 課税所得 | 税率 |
|----------|------|
| 年800万円以下の部分 | 15.0%（軽減税率） |
| 年800万円超の部分 | 23.2% |

**注意**: 軽減税率（15%）の適用期限は2027年3月31日まで延長済み。本則は19%。

#### 2025年度税制改正のポイント

- 所得10億円超の事業年度は、軽減税率が15%→19%に変更
- 通常の不動産賃貸業では影響を受けることは稀

### 2. 地方法人税（国税）

```
地方法人税額 = 法人税額 × 10.3%
```

### 3. 法人住民税（地方税）

#### 法人税割

| 条件 | 税率 |
|------|------|
| 標準税率（一般） | 7.0%（都道府県1.0% + 市町村6.0%） |
| 東京23区（超過税率適用） | 10.4%（資本金1億円超 or 法人税額1,000万円超の場合） |

#### 均等割（赤字でも発生）

| 資本金等 | 従業員数 | 年額（目安） |
|----------|----------|-------------|
| 1,000万円以下 | 50人以下 | 約7万円 |

### 4. 法人事業税（地方税）

**特徴**: 支払った事業年度の損金（経費）に算入可能

#### 東京都23区・中小法人の場合

| 課税所得 | 標準税率 | 超過税率 |
|----------|----------|----------|
| 400万円以下 | 3.5% | 3.75% |
| 400万円超〜800万円以下 | 5.3% | 5.665% |
| 800万円超 | 7.0% | 7.48% |

### 5. 特別法人事業税（国税）

```
特別法人事業税額 = 法人事業税（所得割）額 × 37.0%
```

## 実効税率の計算

### 実効税率の計算式

事業税が翌年度に損金算入されることを考慮した実質的な税負担率：

```
実効税率 = [法人税率 × (1 + 地方法人税率 + 住民税率) + 事業税率 × (1 + 特別法人事業税率)]
         ÷ [1 + 事業税率 × (1 + 特別法人事業税率)]
```

### 標準実効税率（東京都・中小法人）

| 所得区分 | 実効税率（概算） |
|----------|-----------------|
| 800万円以下 | 約21%〜23% |
| 800万円超（標準税率） | 約33.58% |
| 800万円超（超過税率） | 約34.59% |

## 不動産法人の類型

### 1. 不動産所有方式（資産管理会社）

| 項目 | 内容 |
|------|------|
| 概要 | 法人が土地・建物を所有し、入居者から直接家賃を受領 |
| メリット | 所得移転効果が最大、減価償却・借入金利息を最大限活用 |
| デメリット | 物件移転時の登録免許税・不動産取得税が発生 |

### 2. サブリース方式（転貸方式）

| 項目 | 内容 |
|------|------|
| 概要 | 個人所有物件を法人に一括賃貸し、法人が転貸 |
| 利益 | 受取家賃と支払家賃の差額（通常10%〜15%） |
| 注意 | 差額が大きすぎると税務リスクあり |

### 3. 管理受託方式

| 項目 | 内容 |
|------|------|
| 概要 | 個人所有物件の管理業務を法人が受託 |
| 管理料 | 家賃収入の5%〜8%程度 |
| 注意 | 所得移転効果は限定的 |

## 個人 vs 法人の比較

### 税率比較

| 課税所得 | 個人（所得税+住民税） | 法人（実効税率） |
|----------|----------------------|-----------------|
| 400万円 | 約30% | 約21% |
| 800万円 | 約33% | 約23% |
| 900万円 | 約43%（税率33%+住民税） | 約34% |
| 1,800万円 | 約50% | 約34% |
| 4,000万円超 | 約55% | 約34% |

### 法人化の分岐点

- **一般的な目安**: 課税所得900万円〜1,000万円
- **社会保険料を考慮**: 課税所得1,000万円〜1,200万円

### 土地負債利子の取り扱い（重要な差異）

| 項目 | 個人 | 法人 |
|------|------|------|
| 土地取得の借入金利子 | 損益通算の対象外（切り捨て） | 全額損金算入可能 |
| 赤字の繰越期間 | 3年 | 10年 |

## 減価償却の計算

### 法定耐用年数（建物）

| 構造 | 耐用年数 |
|------|----------|
| 鉄筋コンクリート造（RC/SRC） | 47年 |
| 重量鉄骨造（骨格材肉厚4mm超） | 34年 |
| 軽量鉄骨造（肉厚3mm超4mm以下） | 27年 |
| 軽量鉄骨造（肉厚3mm以下） | 19年 |
| 木造 | 22年 |

### 中古資産の耐用年数（簡便法）

**法定耐用年数を全て経過している場合:**
```
耐用年数 = 法定耐用年数 × 0.2（端数切捨て、最低2年）
```

**一部経過している場合:**
```
耐用年数 = (法定耐用年数 - 経過年数) + 経過年数 × 0.2（端数切捨て）
```

### 設備分離の効果

- 建物価格の約20%を建物附属設備として計上可能
- 建物本体（RC: 47年）に対し、設備は15年で償却
- 初期の経費増大による節税効果あり

## 役員報酬の最適化

### 定期同額給与のルール

- 事業年度開始から3ヶ月以内に決定
- 毎月同額を支給（期中の増減は原則損金不算入）

### 所得分散効果

- 配偶者・親族を役員とし、所得を分散
- 各人の所得税率（累進課税）を引き下げ可能

### 社会保険料の負担

- 役員報酬の約30%（法人・個人折半）
- 法人税の節税効果と社会保険料増加のバランスが重要

## 法人維持コスト

### 最低限必要なコスト（年間）

| 項目 | 金額目安 |
|------|----------|
| 法人住民税均等割 | 約7万円（赤字でも発生） |
| 税理士報酬（決算申告のみ） | 15万円〜30万円 |
| 税理士報酬（顧問契約） | 30万円〜60万円 |
| 社会保険料（役員報酬がある場合） | 報酬の約30% |

**役員報酬なしの場合**: 年間約20〜40万円
**役員報酬ありの場合**: +報酬の約30%

## 2025年度税制改正

### 主な改正内容

| 項目 | 内容 |
|------|------|
| 中小法人軽減税率 | 2027年3月31日まで延長 |
| 所得10億円超の場合 | 軽減税率15%→19%に引き上げ |
| 賃上げ促進税制 | 給与増加額の最大45%を税額控除 |

### インボイス制度の影響

| 物件種別 | 影響 |
|----------|------|
| 住宅用 | 非課税売上のため影響なし |
| 事業用（店舗等） | 課税売上となり、消費税納税の可能性あり |

## 申告・納付スケジュール

### 基本スケジュール

| 項目 | 期限 |
|------|------|
| 確定申告期限 | 事業年度終了日の翌日から2ヶ月以内 |
| 中間申告 | 前年度法人税額20万円超の場合、期首から8ヶ月目まで |

### 推奨決算期

- 不動産賃貸業の繁忙期（2月〜3月）を避ける
- 税理士の手が空く夏〜秋（例: 9月決算・11月申告）が一般的

## FAQ（構造化データ対応）

### Q1. 法人化する最大のメリットは何ですか？

A. **税率の頭打ち（最大約34%）と経費範囲の拡大**です。個人の所得税・住民税は最高55%に達しますが、法人は約34%で頭打ちとなります。また、土地の借入金利子も全額経費にできます。

### Q2. 法人化を検討すべきタイミング（年収目安）は？

A. **課税所得900万円**が一般的な分岐点です。社会保険料を考慮すると、実質的なメリットが出るのは**課税所得1,000万円〜1,200万円**からとなるケースが多いです。

### Q3. 役員報酬はいくらに設定すべきですか？

A. 法人に残す利益にかかる法人税と、個人が受け取る給与にかかる所得税・住民税・社会保険料の合計が最小になるポイントを探します。所得税率が跳ね上がるライン（課税所得695万円、900万円）の手前で止めるのがセオリーです。

### Q4. 法人の維持費用は最低いくらかかりますか？

A. 役員報酬なしでも**年間約20〜40万円**（均等割7万円、税理士報酬15〜30万円）。役員報酬を出す場合は、さらにその**約30%の社会保険料**が加わります。

### Q5. 赤字でも税金はかかりますか？

A. はい。**法人住民税の均等割（年間約7万円）**は、赤字であっても必ず支払う必要があります。
