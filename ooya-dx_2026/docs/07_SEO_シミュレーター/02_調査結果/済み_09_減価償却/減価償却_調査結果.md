# 減価償却シミュレーター 調査結果

## 基本情報

- **正式名称**: 減価償却（げんかしょうきゃく / Depreciation）
- **根拠法令**:
  - 所得税法第49条（個人）
  - 法人税法第31条（法人）
  - 減価償却資産の耐用年数等に関する省令
- **管轄省庁**: 国税庁

## 計算式

### 定額法（Straight-Line Method）

```
年間減価償却費 = 取得価額 × 定額法の償却率
```

※平成28年4月1日以降に取得した建物・建物附属設備・構築物は定額法のみ適用

### 中古資産の耐用年数（簡便法）

**ケースA：法定耐用年数を全て経過している場合**
```
耐用年数 = 法定耐用年数 × 0.2（端数切捨て、最短2年）
```

**ケースB：一部経過している場合**
```
耐用年数 = (法定耐用年数 - 経過年数) + 経過年数 × 0.2（端数切捨て、最短2年）
```

### 初年度の月割り計算
```
初年度償却費 = 年間償却費 × (取得月から12月までの月数) / 12
```

## 競合ツール調査

1. **楽待（Rakumachi）** - 不動産投資シミュレーション機能あり。設備分離などの詳細設定は簡易化
2. **CASIO keisan** - 数学的に正確だが不動産特有の機能なし
3. **各税理士事務所サイト** - 解説記事が中心、計算機能は限定的

### 差別化ポイント
- 中古物件の簡便法に完全対応
- 建物構造（鉄骨の肉厚含む）を詳細に選択可能
- 月割り計算の自動化
- 計算式の透明性（弁護士推奨対応）

## 注意点・免責事項に記載すべき内容

- 本計算は概算値であり、実際の金額は異なる場合がある
- 土地は減価償却の対象外である
- 建物と土地の按分方法は売買契約書や固定資産税評価額を参考にする
- 設備分離（建物附属設備の区分）については別途検討が必要
- 平成28年4月1日以降取得の建物等は定額法のみ適用
- 最終的な判断は税理士等の専門家に相談すること

| 項目名 | 型 | 必須 | 単位 | バリデーション |
|--------|-----|------|------|----------------|
| 建物取得価額 | number | ○ | 円 | 0以上 |
| 建物構造 | select | ○ | - | 6種類から選択 |
| 築年数 | number | - | 年 | 0-100 |
| 取得月 | select | - | 月 | 1-12 |

## 出力項目の設計

| 項目名 | 計算式 | 表示形式 |
|--------|--------|----------|
| 年間減価償却費 | 取得価額 × 償却率 | ○○円（強調表示） |
| 初年度償却費 | 年間 × 月数/12 | ○○円 |
| 適用耐用年数 | 簡便法 or 法定 | ○○年 |
| 償却率 | テーブル参照 | ○○% |
