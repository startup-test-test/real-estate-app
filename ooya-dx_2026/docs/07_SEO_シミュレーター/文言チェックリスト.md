# シミュレーター文言チェックリスト

シミュレーターの解説文・説明文を作成する際に確認するガイドラインです。
法的リスクを回避するため、断定的な表現を避け、適切なヘッジ表現を使用してください。

---

## 前提：運営者の状況

**当サイト運営者は以下の資格・免許を保有していません：**
- 税理士資格なし
- 弁護士資格なし
- 宅地建物取引士（宅建）なし
- 司法書士資格なし

この状況で不動産領域のツールを提供するため、以下に該当しないよう注意が必要です。

| 違反リスク | 該当する行為 |
|-----------|-------------|
| **非弁行為**（弁護士法72条） | 法律相談、法的アドバイス、契約書作成の助言 |
| **税理士法違反** | 税務相談、節税アドバイス、確定申告の助言 |
| **宅建業法違反** | 仲介行為、契約締結の助言、重要事項説明 |

**→ 私たちがやっていいのは「情報提供」と「計算ツールの提供」のみ**

## 基本ルール（これだけ覚える）

| ルール | 理由 |
|--------|------|
| **断定はNG** | 「〜です」と言い切らない。「〜とされています」「〜が一般的です」に |
| **FAQ/Q&AはNG** | 質問に断定的に回答する形式は危険 |
| **アドバイスはNG** | 「〜した方がいい」「〜がおすすめ」は法的助言になりうる |
| **数字は「約」をつける** | 「70%」→「約70%」「70%程度」 |
| **免責事項は必須** | 全ページに統一文言を記載 |

## 1. 避けるべき表現（NG例）

### 断定的な表現
| NG | 理由 |
|---|---|
| 〜です。〜ます。（断定） | 法律や税制は個別事情で異なる可能性がある |
| 必ず〜 | 例外がある場合に問題 |
| 絶対に〜 | 同上 |
| 〜しなければならない | 義務として断定している |
| 〜できます | 条件によっては適用されない場合がある |
| 〜が正しい | 判断を断定している |
| 〜すべきです | アドバイスとして断定している |

### 法的アドバイスに該当しうる表現
| NG | 理由 |
|---|---|
| 〜した方がお得です | 具体的なアドバイス |
| 〜をおすすめします | 推奨行為 |
| 〜が最適です | 判断の断定 |
| 〜するべきではありません | 行動の制限を示唆 |

### Q&A・FAQ形式
| NG | 理由 |
|---|---|
| Q: 〜ですか？ A: はい、〜です | 断定的な回答になりやすい |
| よくある質問への回答 | 個別事情を無視した一般論になる |

## 2. 推奨するヘッジ表現（OK例）

### 一般的なヘッジ
| OK表現 | 使用例 |
|---|---|
| 一般的に〜 | 「一般的に買主が負担します」 |
| 〜とされています | 「〜が適用されるとされています」 |
| 〜の場合があります | 「異なる場合があります」 |
| 〜程度 | 「約70%程度」 |
| 約〜 | 「約50万円」 |
| 〜といわれています | 「時価の約70%といわれています」 |
| 〜が多いです | 「買主が負担することが多いです」 |

### 条件付き表現
| OK表現 | 使用例 |
|---|---|
| 〜の場合は | 「自己居住用の場合は軽減されます」 |
| 〜を満たす場合 | 「条件を満たす場合、適用されます」 |
| 原則として〜 | 「原則として〜ですが、例外もあります」 |

### 注意喚起
| OK表現 | 使用例 |
|---|---|
| 詳細は専門家にご確認ください | 必ず記載 |
| 個別の状況により異なります | リスク回避 |
| 概算値です | 計算結果の注釈 |

## 3. セクション別チェックリスト

### タイトル・見出し
- [ ] 「完全ガイド」「徹底解説」など過大な表現を避ける
- [ ] 「〜を教えます」など専門家的な表現を避ける

### 解説文（〜とは）
- [ ] 制度の概要説明に留める
- [ ] 「〜とされています」「〜といわれています」を使用
- [ ] 具体的な法的アドバイスを含まない

### 計算方法の説明
- [ ] 計算式の説明は事実ベースでOK
- [ ] 「一般的に」「約」「程度」を適宜使用
- [ ] 端数処理等の細部は「場合があります」を付ける

### 軽減条件・適用条件
- [ ] 「〜の場合は適用されます」の条件付き表現を使用
- [ ] 「必ず適用される」とは書かない
- [ ] 「詳細は専門家に確認」を促す

### 計算例
- [ ] 「例えば」「仮に」を使用
- [ ] 「この場合」と条件を明示
- [ ] 「概算」「目安」であることを明記

### 早見表
- [ ] 「目安」「概算」であることを明記
- [ ] 注釈で「実際は異なる場合があります」を記載

## 5. 文言レビューチェックリスト

新しいシミュレーターを公開する前に確認：

### 全体
- [ ] FAQ/Q&Aセクションがないこと
- [ ] 断定的な「〜です」が最小限であること
- [ ] ヘッジ表現が適切に使われていること
- [ ] 免責事項が記載されていること

### 具体的な表現
- [ ] 「必ず」「絶対」「すべき」がないこと
- [ ] 「おすすめ」「お得」「最適」がないこと
- [ ] 「一般的に」「程度」「約」が使われていること
- [ ] 「場合があります」「とされています」が使われていること

### 専門家への誘導
- [ ] 「詳細は専門家にご相談ください」の記載があること
- [ ] 計算結果は「概算」「目安」と明記されていること

## 6. 修正例

### Before（NG）
```
登録免許税は買主が負担します。
軽減税率は0.15%です。
この計算で正確な金額がわかります。
```

### After（OK）
```
登録免許税は、一般的に買主が負担することが多いです。
軽減税率は0.15%とされています（令和9年3月末まで）。
この計算結果は概算値であり、実際の金額は異なる場合があります。
```
