## 💡 技術的メモ

### ページタイトル実装のベストプラクティス
```typescript
useEffect(() => {
  document.title = 'ページ名 | サービス名';
  return () => {
    // クリーンアップ（必要に応じて）
  };
}, []);
```

### 考慮事項
- 動的コンテンツ（物件詳細等）では、データ取得後にタイトル更新
- SPAのため、ルート変更時の確実なタイトル更新が重要
- OGPタグとの整合性も今後検討
